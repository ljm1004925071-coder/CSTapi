cst.units package¶

The cst.units package offers methods and classes to work with units supported by CST Studio Suite

Usage¶

from cst.units import mm, um, mil, W, V, A, Unit

# Create quantities with units
l1 = 2 * mm
l2 = 3 * um
l3 = 5 * mil
p1 = 20 * W
i1 = 5 * A
print(l1)               # prints "2 mm"
print(l2)               # prints "3 µm"
print(l3)               # prints "5 mil"
print(p1)               # prints "20 W"
print(i1)               # prints "5 A"

# Create quantities with string based unit, only string variants from the Predefined units table can be used.
alength = 42 * Unit("mm")
aspeed = 120 * Unit("km") / Unit("hour")  # Note that you cannot use "km/h" as it is not one of the predefined units
apower = 55 * Unit("GW")

# Compute derived quantities with automatic unit conversions
l4 = l1 + l2            # add "mm" and "µm" resulting in "mm"
l5 = l2 + l1            # add "µm" and "mm" resulting in "µm"
print(l4)               # prints "2.003 mm"
print(l5)               # prints "2003 µm"
u1 = p1 / i1            # divide "W" by "A" resulting in "V"
print(u1)               # prints "4 V"

# Enforce representation using a specific unit
l6 = l3.convert_to(um)
print(l6)               # prints "127 µm"

# Convert to float without unit
# Warning: Only do this, if the exact unit of the quantity is known.
#          Use "convert_to" to enforce a specific unit.
print(l3.value)         # prints "5"
print(l6.value)         # prints "127"
print(i1.value)         # prints "5"

import random
l7 = random.choice([l3, l6])    # result may use either "mil" or "µm"
print(l7.value)                 # prints value with unknown/random unit
print(l7.convert_to(mm).value)  # prints value with known unit

Reference¶

The cst.units library provides an interface to work with SI and imperial units.

class ComplexQuantity¶

Class to hold a complex-valued physical quantity complex_value * Unit.

convert_to(dest_unit: units.Unit) → units.ComplexQuantity¶

Converts the current value * unit and expresses it in the given dest_unit. Will throw an exception if unit type dest_unit cannot be converted to.

pow(nom: int, denom: int) → units.ComplexQuantity¶

Raises the value and the Unit to the power nom/denom use the ** operator to raise to round integer values.

sqrt() → units.ComplexQuantity¶

Takes the square root of the value and the Unit.

property unit¶

The physical Unit pertaining to the quantity.

property value¶

The value portion of the quantity.

class Quantity¶

Class to hold a real-valued physical quantity value * Unit.

convert_to(dest_unit: units.Unit) → units.Quantity¶

Converts the current value * unit and expresses it in the given dest_unit. Will throw an exception if unit type dest_unit cannot be converted to.

pow(nom: int, denom: int) → units.Quantity¶

Raises the value and the Unit to the power nom/denom use the ** operator to raise to round integer values.

sqrt() → units.Quantity¶

Takes the square root of the value and the Unit.

property unit¶

The physical Unit pertaining to the quantity.

property value¶

The value portion of the quantity.

class Unit(predef_unit: str)¶

Class representing a physical unit.

Gets the Unit given by the string if it exists. E.g. ‘mm’, ‘kg’, ‘MV’.

static decode(arg0: str) → units.Unit¶

Deserializes the given string to a Unit

encode() → str¶

Serializes the Unit to a string

get_symbol() → str¶

Returns the Unit

inSI() → units.Unit¶

Returns the equivalent quantity expressed in strict SI-units.

pow(nom: int, denom: int) → units.Unit¶

Raises the value and the Unit to the power nom/denom use the ** operator to raise to round integer values.

simplify() → units.Unit¶

Tries to simplify the value and unit.

convert_value(value: object, from_unit: units.Unit, to_unit: units.Unit) → object¶

Converts a value expressed in from_unit to to_unit

scaling_factor_to_SI(unit: Unit)¶

Compute scaling factor into equivalent SI unit.

Parameters:

unit – Simple or compound unit.

Returns:

numerical scaling factor for converting into SI units.
