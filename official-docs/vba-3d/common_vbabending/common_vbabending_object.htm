Bending Object

This object is used to bend thin planar sheets or shapes on a solid shape.

General Methods

ParamSweepAndOptimizerChecksResult( bool switch )

To activate a check routine performed during a parameter sweep or optimizer calculation. The check verifies whether the sheets are entirely bent on the solid. The check is performed for bend shape and bend layer stackup operations.

Reset

Resets all internal settings to their default values.

 

Bend Shape

Sheet ( name sheetname )

Sets the name of the sheet to bend.

 

Solid ( name solidname  )

Sets the name of the solid to bend on.

 

Faces ( name list_of_face_ids )

Sets the list of faces to bend on. The faces are specified by the face ids. For example: "10,9,8,7".

 

Bend

Performs the bending operation.

 

Bend Layerstackup

Stackup ( name stackupname )

Sets the name of the folder which will be used for the layer stackup bending. The shapes in this folder are used for the bending operation. This can be a material or group folder. For example: "material$Folder1".

 

Shape ( name shapename )

Sets the name of the shape which will be used for the layer stackup bending. Several shape names can be used. You may not use the commands Stackup and Shape at the same time.

 

Solid ( name solidname  )

Sets the name of the solid to bend on.

 

Faces ( name list_of_face_ids )

Sets the list of faces to bend on. The faces are specified by the face ids. For example: "10,9,8,7".

 

FlexBending ( bool switch )

This flag is only used by a layer stackup bending. The default is "True" and means that the length of the layers will be adapted flexible such that they match together after the bending.

 

Bend

Performs the layer stackup bending operation.

 

Cylindrical Bending

Stackup ( name stackupname )

Sets the name of the folder which will be used for the cylindrical bending. The shapes in this folder are used for the cylindrical bending operation. This can be a material or group folder. For example: "material$Folder1".

 

Shape ( name shapename )

Sets the name of the shape which will be used for the cylindrical bending. Several shape names can be used. You may not use the commands Stackup and Shape at the same time.

 

Centralized ( bool onesided )

Sets the impact direction of the cylindrical bending operation. False indicates a one-sided direction which means that only the half-space of the u-axis will be bent. True indicates that the bending is performed along the entire u-axis.

 

Angle ( double angle )

Sets the angle of the cylinder around which the items are going to be bent. Either the angle or the radius must be zero.

 

Radius ( double radius )

Sets the radius of the cylinder around which the items are going to be bent. Either the angle or the radius must be zero.

 

ULength ( double length )

Optionally, the u-length can be specified to limit the bending region along the u-axis of the local WCS.

 

VLength ( double length )

Optionally, the v-length can be specified to limit the bending region along the v-axis of the local WCS.

 

ReferenceSolid ( name shapename )

Sets the name of the reference solid. The reference solid is used internally to determine whether unconnected items shall be bent. If the reference solid is not specified then the algorithm determines an internally reference solid by uniting all items which shall be bent. This operation can be very time-consuming therefore it is recommended to use a reference solid when many shapes shall be bent.

 

FlexBend

Performs the cylindrical bending operation around a virtual cylinder defined by either an angle or an radius. The position and orientation of the cylinder is determined by setting the local WCS.

 

Wrap Sheet

Sheet ( name sheetname )

Sets the name of the sheet to wrap.

 

Solid ( name solidname  )

Sets the name of the target solid on which to wrap the sheet or sheets.

 

Faces ( name list_of_face_ids )

Sets the list of faces of the target solid which are used to wrap the sheet or sheets on. The faces are specified by the face ids. For example: "10,9,8,7".

 

WrapDirection ( name direction )

Sets the direction of the wrap operation. The sheet or sheets can be wrapped in u or v direction of the target solid. It is recommended to choose the direction in which the target shape is stronger curved.

 

Wrap

Performs the wrap operation.

 

Default Settings

Sheet ("")

Stackup ("")

Shape ("")

Solid ("")

Faces ("")

FlexBending ("True")

Centralized ("False")

ReferenceSolid ("")

WrapDirection ("")

EmbedInTarget ("False")

Example

With Bending

.Reset

.Sheet "component1:sheet"

.Solid "component2:solid1"

.Faces "10,9,8,7"

.Bend

End With

 

With Bending

.Reset

.Shape "component1:solid1"

.Shape "component1:solid2"

.Solid "component1:solidToBendOn"

.Faces "10,9,8,7"

.FlexBending "True"

.Bend

End With

 

With Bending

.Reset

.Stackup "material$Folder1"

.Solid "component1:solidToBendOn"

.Faces "10,9,8,7"

.FlexBending "True"

.Bend

End With

 

With Bending

     .Reset

     .Shape "component1:solid1"

     .Centralized "False"

     .Angle "0"

     .Radius "10"

     .ULength "3"

     .VLength "5"

     .FlexBend

End With

 

With Bending

.Reset

.Sheet "component1:sheet"

.Solid "component2:solid1"

.Faces "7"

.WrapDirection "v"

.Wrap

End With
