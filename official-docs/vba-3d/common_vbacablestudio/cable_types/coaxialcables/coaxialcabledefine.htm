CoaxialCableDefine

Define or modify a coaxial cable. For the inner cable definition please refer to the SingleWireDefine command.

For hierarchical elements like screens please refer to Hierarchical structure definition.

  • Methods

  • Keywords

  • Default Settings

  • Examples

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string value)

Sets the name of the wire that will be edited or created in ’value’

Parameter 'objType' is "CSCoaxialCable".

Parameter 'cmdId' is "Name".

AddStringArgument (string name, string value)

Sets the string value for the parameter specified by 'name'.

For possible values please refer to the details below.

AddArgument (string name, double value)

Sets the double value for the parameter specified by 'name'.

For possible values please refer to the details below.

Screen definition

Screen

 

.AddStringArgument "LocalUnits", "<enum{m, cm, mm, um, nm, ft, in, mil}>"

.AddStringArgument("Screen", string name)

.AddStringArgument("Screen$Index", string index)

.AddArgument(""Screen$frequency_val", double f)

.AddStringArgument("Screen$frequency_ext", string freq)

.AddStringArgument("Screen$screen$shape", string shapeDefinition)

shapeDefinition: as in SingleWire

.AddStringArgument("Screen$screen$material", string materialId)

.AddArgument("Screen$screen$wrap_thickness", double t)

.AddStringArgument("Screen$screen_model_type", string type)

type: One of SimplifiedModel, SolidModel, BraidedModel, FoilLongitudinalModel, FoilSpiralModel

BraidedModel

 

.AddStringArgument("Screen$kley$calculation_screen_model_type", string type)

type: One of Analytic, Measured

.AddStringArgument("Screen$kley$defined_by", string type)

type: one of picks_per_unit, optical_coverage, braind_angle

.AddArgument("Screen$kley$optical_coverage", double c)

.AddArgument("Screen$kley$braid_angle", double a)

.AddArgument("Screen$kley$picks_per_unit", double ppu)

Provide one of the parameters depending on the value for defined_by above

.AddArgument("Screen$kley$number_of_carriers", string numCarriers)

numCarriers: number of groupings for filaments

.AddStringArgument( "Screen$kley$carrier_defined_by", string type)

type: Either Tape or Strand used to construct the braid

.AddArgument("Screen$kley$filament_diameter", double d)

d: diameter of a single individual strand for the screen

.AddArgument("Screen$kley$number_of_filaments", string numFilaments)

numFilaments: number of filament groups if individual strands

.AddArgument ("Screen$kley$tape_width", double w)

w: width of the tape used in the braid

.AddArgument ("Screen$kley$tape_thickness", double t)

t: thickness of the tape used for the braid
