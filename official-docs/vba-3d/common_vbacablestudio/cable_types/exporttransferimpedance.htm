ExportTransferImpedance

Exports the impedance data of a given cable of given type to a file.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

AddStringArgument (string name, string value)

Sets the 'value' for parameters listed under keywords below.

AddUnitArgument(string propertyId, double value, string ext)

propertyId: MaxFrequency for the upper limit of the frequency. Starting frequency is 0.

value: the numerical base value, e.g. 1000

ext: SI unit name like Milli, Mega, Kilo. (none) for values without SI extension

Execute(string command)

Sets the parameters for the calculation and writes the results to the specified file.

Keywords

FileName: the full path for the file to create. Must not be a directory name and will be overwritten.

CableName: Name of the cable from the given CableType section of the cable library

CableType: One of 'Coaxial' or 'Group'. These are the only types that can have screens.

ImpedanceType: One of 'SelfImpedance' or 'TransferImpedance'

Logarithmic: boolean value to calculate the results either linear (value = 'false') or logarithmic (value = 'true'

ScreenName: One or more screen names of the given cable. Corresponds to the names in the GUI, so. e.g. 'Screen_1'

Samples: Specifies the positive number of samples to calculate, default 500.

Examples

 With CableStudio

    .Reset

    .AddStringArgument("FileName","e:\tmp\selfimpedance.cir")

    .AddStringArgument("CableName","MyGroup")

    .AddStringArgument("CableType","Group")

    .AddStringArgument("ImpedanceType","SelfImpedance")

    .AddStringArgument("Logarithmic","true")

    .AddStringArgument("ScreenName","Screen")

    .AddStringArgument("ScreenName","Drain1")

    .AddStringArgument("ScreenName","Screen_1")

    .AddStringArgument("Samples","500")

    .AddUnitArgument("MaxFrequency","1000", "Mega")

    .Execute "ExportTransferImpedance"

End With
