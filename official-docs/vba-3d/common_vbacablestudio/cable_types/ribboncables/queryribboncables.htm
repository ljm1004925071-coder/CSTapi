QueryRibbonCables

Allows to retrieve information about ribbon cable entries.

  • Queries

  • Example

Queries

To work with elements out of a list of ribbon cables there is the QueryRibbonCables command with the given default parameters.

Examples

 

Dim bVal As Boolean

Dim sArray() As String

 

With CableStudio

  .Reset

  .AddStringArgument("name", "Flat")

  .AddStringArgument("param", "item_name_exists")

  bVal = .GetBool("QueryRibbonCables")

End With

 

With CableStudio

  .Reset

  .AddStringArgument("param", "item_names")

  sArray = .GetStringArray("QueryRibbonCables")

End With

 

Alternative using a Variant type:

 

Dim nResults As Integer

Dim sVArray As Variant

 

With CableStudio

  .Reset

  .AddStringArgument("param", "item_names")

  nResults = .GetArray("QueryRibbonCables", sVArray)

End With
