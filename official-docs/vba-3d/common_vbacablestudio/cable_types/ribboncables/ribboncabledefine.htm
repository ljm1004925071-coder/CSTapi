RibbonCableDefine

Defines or modifies a ribbon cable

Additional parameters (conductor/insulator) for the command are described in SingleWireDefine.

  • Methods

  • Examples

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string value)

Parameter 'objType' is "CSRibbonCable".

Parameter 'cmdId' is "Name".

Parameter 'value' sets the name of the cable that will be edited or created.

AddArgument (string name, double value)

Provide additional attributes to define a ribbon cable

Double values for ’name’:

  Distance: distance between the internal wires, with corresponding 'value'>

  Strength: thickness of the bridge between internal wires, with corresponding 'value'

  Number: number of sub cables in the ribbon, 'value' determines the number of used wires

Execute (string command)

Create a new ribbon cable element with the given parameters or defaults.

Default settings

.AddStringArgument "LocalUnits", "<enum{m, cm, mm, um, nm, ft, in, mil}>"

.AddArgument "Number", "2"

.AddArgument "Distance", "1.0"

.AddArgument "Strength", "1.0"

Examples

With CableStudio

    .Reset

    .AddStringArgument "LocalUnits", "mm"

    .OutputName "CSRibbonCable", "Name", "Ribbon_Cable_1"

    .Execute "RibbonCableDefine"

End With

 

With CableStudio

    .Reset

    .AddStringArgument "LocalUnits", "mil"

    .AddArgument "Number", "3"

    .AddArgument "Distance", "118.0"

    .AddArgument "Strength", "118.0"

    .Execute "RibbonCableDefine"

End With
