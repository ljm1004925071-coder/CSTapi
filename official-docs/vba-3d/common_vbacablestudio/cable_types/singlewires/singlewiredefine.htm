SingleWireDefine

Creates a single wire cable complete with geometries

  • Methods

  • Specific shapes

  • Default settings

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string value)

Parameter 'objType' is "CSSingleWire".

Parameter 'cmdId' is "Name".

Parameter ’value’ defines the name of the wire that will be edited or created.

AddArgument (string name, double value)

Sets the value for the parameters of type 'name'. Default value ”0.0”.

For possible values please refer to the details below.

AddStringArgument ( string name,  string value)

Sets the value for the parameters of type 'name'.

For possible values please refer to the details below.

Execute (string command)

Create a new single wire element with the given parameters or defaults.
