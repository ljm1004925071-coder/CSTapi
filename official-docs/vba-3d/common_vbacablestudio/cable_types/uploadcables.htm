UploadCables

Upload all current project cables to the default cable library.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

AddStringArgument (string name, string value)

Sets the 'value' for parameter 'Handle Duplicates' as one of 'Replace', 'Add'.

Execute(string command)

Exports all current project cables to the default cable library.
Existing cables in the library are either replaced or added with a unique name (_1, _2,...).

 

Examples

With CableStudio

    .Reset

    .AddStringArgument "HandleDuplicates", "Replace"

    .Execute "UploadCables"

End With
