ConnectionSegmentBundle

Creates or modifies a connecting segment bundle definition between a Cable Studio cable segment and a 3D structure

 

For naming conventions and the idea behind hierarchical segment elements like cables, bundles and sub-cables please refer to Hierarchical structure definition.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string name)

Sets the ’name’ of the connecting segment that will be edited or created.

objType ’CSConnectionSegment’, cmdID ’Name’ defines the name for this connecting segment.

AddInputName(string objType, string elementId, string nodeId)

objType ”CSNode”: with elementId "Name" and nodeID the name of an existing Cable Studio node.

objType ’CSBundle’: with elementId "Name" and nodeID the name of an existing Cable Studio cable bundle

objType ’CSSingleWire’: with elementId "connection segment name$sub cable name$CableTypeID" and nodeID the name of a valid cable library single wire cable

AddStringArgument(string elementId, bool value)

elementId "<Bundle_name>$RandomBundle", value "<boolean>": bundle is to be rebundled in meshing step

elementId "<Bundle_name>$PressedIn", value "<boolean>": bundle content is pressed in (typically for drain wires)

AddStringArgument(string elementId, string value)

elementId "<Bundle_name>$Cable", value 'sub cable id' which sub cable of the bundle is used for this operation

elementId "<Bundle_name>$<Sub cable id>", value 'SegmentBundleElement': declares the type of this bundle.

elementId "<Bundle_name>$CableType", value is one of CSSingleWire or CSDefaultCableType. If CSSingle wire the CSSingleWire property above is defined.

elementId "<Bundle_name>$Index", value is the sub cable ID. Provides an order of the contained sub cables

AddArgument(string elementId, double value)

elementId "<connection segment name>$<cable bundle inner cable name>$x", value X position in the bundle

elementId "<connection segment name>$<cable bundle inner cable name>$y", value Y position in the bundle

elementId "<connection segment name>$<cable bundle inner cable name>$rotation", value rotation in the bundle

Execute(string command)

Create or modify a connecting segment, command ”ConnectionSegmentBundle”

Example

With CableStudio

    .Reset

    .AddInputName "CSBundle", "Name", "B"

    .AddInputName "CSNode", "Name", "N1"

    .OutputName "CSConnectionSegment", "Name", "B_N1_to_3D_11"

//the part bellow is identical to SegmentBundle VBA

    .AddStringArgument "B_N1_to_3D_11$RandomBundle", "true"

    .AddStringArgument "B_N1_to_3D_11$PressedIn", "false"

    .AddStringArgument "B_N1_to_3D_11$Cable", "RC_4_2"

    .AddStringArgument "B_N1_to_3D_11$RC_4_2", "SegmentBundleElement"

    .AddArgument "B_N1_to_3D_11$RC_4_2$x", "-0.0457784"

    .AddArgument "B_N1_to_3D_11$RC_4_2$y", "-1.7738552"

    .AddArgument "B_N1_to_3D_11$RC_4_2$rotation", "59.7216797"

    .AddStringArgument "B_N1_to_3D_11$RC_4_2$CableType", "CSSingleWire"

    .AddInputName "CSSingleWire", "B_N1_to_3D_11$RC_4_2$$CableTypeId", "LIFY_0qmm75"

    .AddStringArgument "B_N1_to_3D_11$RC_4_2$Index", "0"

    .AddStringArgument "B_N1_to_3D_11$Cable", "RC_4_1"

    .AddStringArgument "B_N1_to_3D_11$RC_4_1", "SegmentBundleElement"

    .AddArgument "B_N1_to_3D_11$RC_4_1$x", "0.1427861"

    .AddArgument "B_N1_to_3D_11$RC_4_1$y", "0.8317136"

    .AddArgument "B_N1_to_3D_11$RC_4_1$rotation", "160.4443359"

    .AddStringArgument "B_N1_to_3D_11$RC_4_1$CableType", "CSDefaultCableType"

    .AddStringArgument "B_N1_to_3D_11$RC_4_1$Index", "1"

    .AddStringArgument "B_N1_to_3D_11$Cable", "RC_3_2"

    .AddStringArgument "B_N1_to_3D_11$RC_3_2", "SegmentBundleElement"

    .AddArgument "B_N1_to_3D_11$RC_3_2$x", "-1.9674747"

    .AddArgument "B_N1_to_3D_11$RC_3_2$y", "-0.330295"

    .AddArgument "B_N1_to_3D_11$RC_3_2$rotation", "1.6809082"

    .AddStringArgument "B_N1_to_3D_11$RC_3_2$CableType", "CSDefaultCableType"

    .AddStringArgument "B_N1_to_3D_11$RC_3_2$Index", "2"

    .Execute "ConnectionSegmentBundle"

End With
