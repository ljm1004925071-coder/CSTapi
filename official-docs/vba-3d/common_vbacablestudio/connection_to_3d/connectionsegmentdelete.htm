ConnectionSegmentDelete

Deletes an existing connecting segment between a Cable Studio cable bundle and a 3D structure.

 

For naming conventions and the idea behind hierarchical segment elements like cables, bundles and sub-cables please refer to Hierarchical structure definition.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

AddInputName(string objType, string elementId, string nodeId)

objType ”CSNode”: with elementId "Name" and nodeID the name of an existing Cable Studio node.

objType ’CSBundle’: with elementId "Name" and nodeID the name of an existing Cable Studio cable bundle

objType ’CSConnectionSegment’: with elementId "Name" and nodeID the name of an existing Cable Studio connection segment, one or more.

Execute(string command)

Delete the given connecting segment(s), command ”ConnectionSegmentDelete”

Example

With CableStudio

    .Reset

    .AddInputName "CSBundle", "Name", "B"

    .AddInputName "CSNode", "Name", "B$N1"

    .AddInputName "CSConnectionSegment", "Name", "B$N1$B_N1_to_3D_11"

    .Execute "ConnectionSegmentDelete"

End With
