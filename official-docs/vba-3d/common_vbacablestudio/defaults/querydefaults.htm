QueryDefaults

Allows to retrieve information about default settings.

  • Queries

  • Example

Queries

To access the individual default settings the QueryDefaults command is available with the following possible values for param.

 

Default Property Parameter value Returns
Default conductor material conductor_material String
Default environment material environment_material String
Default insulator material insulator_material String
Default screen material screen_material String
Default junction material junction_material String
Default junction radius junction_radius Double
Default connection wire connection_wire String
Default cross section units cross_section_units String

 

Examples

 

Dim sRes As String

Dim dRes As Double

Dim bRes As Boolean

 

With CableStudio

  .Reset

  .AddStringArgument("param", "screen_material")

  sRes = .GetString("QueryDefaults")

End With

 

With CableStudio

  .Reset

  .AddStringArgument("param", "junction_radius")

  dRes = .GetDouble("QueryDefaults")

End With
