CSJunctionDefine

Create or modify a shortcut between pins of terminals at nodes.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string value)

Parameter 'objType' is 'CSJunction'.

Parameter 'cmdId' is 'Name'.

Parameter 'value' defines the name of the shortcut that will be edited or created.

AddStringArgument (string name, string value)

Parameter 'name' can be the following:

  "type" with value one of SHORT, RESISTOR.

  "node" with value the name of the node the following terminals are defined for.

  "<node name>" with value the name of the terminal, e.g. 'N1_RC_2_1'.

  "<node name>$ref" with value the name of the reference node to create.

Note: At least 2 node terminals or one node terminal and one ref must be provided..

Execute (string command)

Create or modify a shortcut or resistor between cable terminals at terminal nodes with the given parameters or defaults.

Example

With CableStudio

    .Reset

    .OutputName "CSJunction", "Name", "Junc"

    .AddStringArgument "type", "SHORT"

    .AddStringArgument "node", "N1"

    .AddStringArgument "N1", "N1_RC_2_1"

    .AddStringArgument "N1", "N1_RC_2_2"

    .AddStringArgument "N1$ref", "N1_REF"

    .Execute "CSJunctionDefine"

End With
