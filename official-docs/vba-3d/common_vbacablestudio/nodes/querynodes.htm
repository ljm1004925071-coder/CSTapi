QueryNodes

Allows to retrieve information about node entries.

  • Queries

  • Example

Queries

To work with elements out of a list of Nodes there is the QueryNodess command with the given default parameters.

To access the individual properties of a node the QueryNodesProperty command is available with the following possible values for param.

 

Nodes Property Parameter value Returns
Number number_of_items Long value with number of nodes in the project

 

Examples

 

Dim nResults As Long

 

With CableStudio

  .Reset

  .AddStringArgument("param", "number_of_items")

  nResults = .GetLong("QueryNodes")

End With

 

Dim dArray() As Double

With CableStudio

  .Reset

  .AddStringArgument("name", "N1")

  .AddStringArgument("param", "location")

  dArray = .GetDoubleArray("QueryNodeProperty")

End With

 

Alternative using a Variant type:

 

Dim dVArray As Variant

With CableStudio

  .Reset

  .AddStringArgument("name", "N1")

  .AddStringArgument("param", "location")

  nResults = .GetArray("QueryNodeProperty", dVArray)

End With
