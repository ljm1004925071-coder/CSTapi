Query parameters

This chapter provides an overview of the commands available for CST Cable Studio queries of object properties.

Any indexparameter of the following commands is zero-based (i.e. 0 for the first item).

Any property query for a given type requires a “name” string argument to specify the name of the queried object.

 

Queries have one or more parameters to specify the value to be retrieved. They are available for the following Cable Studio elements:

 

Common properties for all queries of stacks (Nodes, Signals, Materials,...):

 

Parameter name Meaning Returns
name Unique name of the item in question String
number_of_items How many items of this type are present Long
item_names List of the names of the items When called with GetStringArray(), returns an array of string objects.
When called with GetArray(), returns a Long value (indicating the number of array elements) and the fills the parameter array of type Variant (see example below)
item_name_exists Checks for the existence of the given name in the current stack Boolean
item_name_from_index Returns the nth item in the stack, n >= 0 String

 

 

Queries can return the following types:

 

Data type Method
Boolean GetBool()
Long GetLong()
Double GetDouble()
String GetString()
Long array GetLongArray()
Double array GetDoubleArray()
String array GetStringArray()
Bool array GetBoolArray()
Long and Array of type Variant as parameter GetArray()
Returns Long value (indicating the number of array elements) and the fills the parameter array of type Variant (see example below

 

Array parameters from GetArray() can be extracted and used similar to this:

 

Dim sArray() As String

With CableStudio

  .Reset

  .AddStringArgument("param", "item_names")

  sArray = .GetStringArray("QueryMaterials")

End With

 

An alternative is code like this:

 

Dim nResults As Long

Dim sArray As Variant

 

With CableStudio

  .Reset

  .AddStringArgument("param", "item_names")

  nResults = .GetArray(<SomeCommand>, sArray)

End With

 

ReDim Preserve sNames(nResults) As String

Dim N As Long

For N = 0 To nResults - 1

  sNames(N) = CStr( sArray( N ) )

Next
