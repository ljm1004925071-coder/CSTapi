CalculateImpedance

This command is an interface to the CST Cable Studio Impedance Calculator. It lets the user calculate the basic modes of the given segment at the given position.

  • Methods

  • Examples

Methods

Reset

Resets all internal settings for this command to their default values.

AddStringArgument(string cmdId, string value)

 

Parameter cmdId Description
Name This is a required parameter that defines the segment the calculator shall work on.
Position This is a required parameter that defines the position along the segment 'A - B' (starting at A with 0 and 100 at B)
LocalFrequency If set to true the default frequencies for the current simulation can be overridden.
Default: false
MinimumFrequency If 'LocalFrequency' is set to true this defines the starting frequency for the simulation.
Default: 1.0M
MaxmumFrequency If 'LocalFrequency' is set to true this defines the end frequency for the simulation.
Default: 1.0G
LogarithmicSweep This parameter determines whether the values are plotted logarithmically or not.
Default: false
Samples This parameter determines the number of intermediate points that are to be used for the simulation.
Default: 100
OhmicLosses This parameter determines whether ohmic losses shall be considered for the simulation.
Default: true
DielectricLosses This parameter determines whether dielectric losses shall be considered for the simulation.
Default: false
SolverAccuracy This parameter defines the accuracy with which the solver will perform its calculations. Higher values will result in longer run time.
Possible values: normal, medium, high, very high
Default: normal
TwistAccuracy This parameter determines the accuracy for twisting cables in the current segment. Higher values will result in longer run time.
Possible values: normal, medium, high, very high
Default: normal
ZeroDFrequency One or more frequencies for which a 0D result shall be created.
CreateEigenmode If set to true and an OutputFolder value was defined then a CSV file containing Eigenmode vectors for the signals used in the simulation.
Default: true
CreateTouchstone If set to true and an OutputFolder value was defined then a Touchstone file matching the impedance network will be created in the given folder.
Default: true

 

 

Execute (string command)

Performs a calculation of the impedance of the given segment at the given position. The results are calculated base mode characteristic curves for each given input set. If defined, 0D and Touchstone files will be generated as well.

Examples

 

With CableStudio

    .Reset

    .addStringArgument("Name","N7 - N12")

    .addStringArgument("Position","12")

    .Execute "CalculateImpedance"

End With

 

With CableStudio

    .Reset

    .addStringArgument("Name","N1 - N2")

    .addStringArgument("Position","75")

    .addStringArgument("LocalFrequency","true")

    .addStringArgument("MinimumFrequency","1M")

    .addStringArgument("MaximumFrequency","1G")

    .addStringArgument("LogarithmicSweep","false")

    .addStringArgument("Samples","250")

    .addStringArgument("OhmicLosses","true")

    .addStringArgument("DielectricLosses","false")

    .addStringArgument("SolverAccuracy","high")

    .addStringArgument("TwistAccuracy","very high")

    .addStringArgument("OutputFolder","d:\SimulationData\Myproject\ImpedanceOutput")

    .addStringArgument("CreateZeroD","true")

    .addStringArgument("ZeroDFrequency","200M")

    .addStringArgument("ZeroDFrequency","400M")

    .addStringArgument("ZeroDFrequency","600M")

    .addStringArgument("ZeroDFrequency","800M")

    .addStringArgument("CreateTouchstone","true")

    .addStringArgument("CreateEigenmode","true")

    .Execute "CalculateImpedance"

End With
