SegmentBundle

Create or modify a segment bundle setup. Wire positions as well as sub bundles wrapped in insulators can be defined with this command.

For naming conventions and the idea behind hierarchical elements like bundles and sub-cables please refer to Hierarchical structure definition.

  • Methods

  • Example

Methods

Reset

Resets all internal settings for this command to their default values.

OutputName(string objType, string cmdId, string name)

Parameter 'objType' is "CSSegment".

Parameter ’cmdId' is "Name".

Parameter 'value' specifies the name of the segment to be created or modified.

AddInputName(string objType, string cmdId, string value)

Parameter 'objType' is one of one of CSSingleWire, CSCoaxialCable, CSRibbonCable, CSTwistedCable, CSCableGroup.
  This parameter is related to the CableType argument below and specifies the element from the cable library to use.

Parameter 'cmdId' specifies the 'CableTypeID' for the cable library element to use.

AddStringArgument(string cmdId, string value)

The hierarchical parameter 'cmdId' may end with the following keywords:
 "RandomBundle" with string 'true' or 'false' indicates whether the segment bundle layout may be altered by the cable mesher.

    "Cable" The name of a cable element to be defined in the current bundle.

    "CableType" one of CSSingleWire, CSCoaxialCable, CSRibbonCable, CSTwistedCable, CSCableGroup.

    "Index" represents the relative position of the cable towards others on the same hierarchy level.

    "Bundle" defines the name of a segment bundle which may hold cables or other bundles.

    <Parent bundle name> The value may be 'SegmentBundle'.

    "Insulator" The value specifies a new insulator element for a Bundle element.

    "shape" This specifies the type of shape the Insulator element may possess: See Shape types.

    "material" The value of this keyword specifies a valid material id for an insulator.

    <cable name> The value of this keyword specifies the type of element. For segments this is 'SegmentBundleElement'.

AddArgument(string cmdId, double value)

Parameter 'cmdId' describes the hierarchical state of the element to define with the following variants:

    "x", "y" or "z"  The value defines the relative x, y or z position of the element in question.

    "rotation" The value defines the rotation of the element in question.

    "wrap_thickness" (or the corresponding parameter from the current shape type)

Execute (string command)

Create or modify a segment element with the given parameters.

Example

With CableStudio

    .Reset

    .OutputName "CSSegment", "Name", "N1 - N2"

    .AddStringArgument "N1 - N2$RandomBundle", "false"

    .AddStringArgument "N1 - N2$Cable", "SW_1"

    .AddStringArgument "N1 - N2$SW_1", "SegmentBundleElement"

    .AddArgument "N1 - N2$SW_1$x", "0.1232043"

    .AddArgument "N1 - N2$SW_1$y", "-1.6019874"

    .AddArgument "N1 - N2$SW_1$rotation", "109.4348145"

    .AddStringArgument "N1 - N2$SW_1$CableType", "CSSingleWire"

    .AddInputName "CSSingleWire", "N1 - N2$SW_1$CableTypeId", "LIFY_0qmm10"

    .AddStringArgument "N1 - N2$SW_1$Index", "0"

    .AddStringArgument "N1 - N2$Bundle", "N1 - N2_Bundle_1_1"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1", "SegmentBundle"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$x", "0.5024518"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$y", "0.9142455"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$rotation", "255.7727051"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$RandomBundle", "false"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Insulator", "N1 - N2$N1 - N2_Bundle_1_1$Insulator"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Insulator$Index", "0"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Insulator$shape", "Wrap"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Insulator$material", "PE"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$Insulator$wrap_thickness", "0.5"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Cable", "SW_2"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2", "SegmentBundleElement"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2$x", "0.0"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2$y", "0.0"

    .AddArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2$rotation", "296.2133789"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2$CableType", "CSSingleWire"

    .AddInputName "CSSingleWire", "N1 - N2$N1 - N2_Bundle_1_1$SW_2$CableTypeId", "LIFY_0qmm75"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$SW_2$Index", "0"

    .AddStringArgument "N1 - N2$N1 - N2_Bundle_1_1$Index", "1"

    .Execute "SegmentBundle"

End With
