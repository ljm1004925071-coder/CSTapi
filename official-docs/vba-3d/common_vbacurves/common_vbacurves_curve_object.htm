Curve Object

This object is used to apply operations on curves and curve items.

  • Methods

  • Example

Methods

NewCurve ( name curvename )

Defines and names a new curve object.

 

DeleteCurve ( name curvename )

Deletes a specified curve object.

 

RenameCurve ( name oldname, name newname )

Renames a specified curve object.

 

DeleteCurveItem ( name curvename, name curveitemname )

Deletes a specified item object associated to a curve object.

 

RenameCurveItem ( name curvename, name oldcurveitemname, name newcurveitemname )

Renames a specified item object associated to a curve object.

 

DeleteCurveItemSegment ( name curvename, name curveitemname, int edgeid )

Deletes a segment of an item object associated to a curve object. The segment is specified by the name of the curve and the item that it belongs as well as an identity number.

 

MoveCurveItem ( name curveitemname, name oldcurvename, name newcurvename )

Moves a specified item from the curve it belongs to another curve object.

 

StartCurveNameIteration ( string type ) long

Start iterating over the curves and returns the number of curves:

"all"

All curves

"open"

Only the open curves

"closed"

Only the closed curves

 

GetNextCurveName () name

Returns the next curve name. Use StartCurveNameIteration to start the iteration.

 

GetPointCoordinates (string curveitemname,  string pid, double &x,double &y,double &z )  bool

Retrieves the coordinates of the point with the given pid. Returns false if there is no such point.

 

SampleCoordinates (string curveitemname,  double dSampleValue, double &x,double &y,double &z )  bool

Retrieves the coordinates of a position on the curve at a normed sample value. Valid sample values are in the range from 0 to 1:

0

start point

0.5

middle position

1

end point

Returns false if there is no such point.

 

GetNumberOfPoints (string curveitemname ) int

Returns the maximum number of points.

 

IsClosed (string curveitemname ) bool

Returns whether it is a closed curve or not.

 

IsCurve ( name name ) bool

Returns whether the specified name is a curve or not.

 

IsCurveItem ( name name ) bool

Returns whether the specified name is a curve item or not.

 

GetAllCurveNames ( name curvename ) string

Returns a comma separated string, which contains curve names. If curvename is empty, the string contains all curve names of the model, otherwise it will contain all curve names of  the specified curve.
Sample code can be seen below in section Example.

 

GetAllCurveItemNames ( name curvename ) string

Returns a comma separated string, which contains curve item names. If curvename is empty, the string contains all curve item names of the model, otherwise it will contain all curve item names of  the specified curve.
Sample code can be seen below in section Example.

Example 1 - Rename and delete

Curve.RenameCurve "curve1", "MyCurves"

Curve.DeleteCurveItemSegment "curve1", "rectangle1", "1"

Example 2 - How to get all curve names, and how to iterate them

Dim sCurvesWithComma   As String

Dim sCurves()          As String

Dim iCount             As Integer

sCurvesWithComma = Curve.GetAllCurveNames("")

sCurves = Split(sCurvesWithComma,",")

For iCount = 0 To UBound(sCurves)

   MsgBox(sCurves(iCount))

Next iCount

Example 2 - Sample curve coordinates in 11 samples (0-10)

Sub Main

Dim sCurve As String

Dim dSample As Double

Dim i,nMaxNodes As Integer

sCurve = "curve1:spline1"

dSample = 0.1

nMaxNodes = (1/dSample)+1

Dim dX() As Double ' x-coordinate

Dim dY() As Double ' y-coordinate

Dim dZ() As Double ' z-coordinate

ReDim nodes(nMaxNodes, 3) As Double

 

For i = 1 To nMaxNodes

   dSample = CDbl(i-1) / (nMaxNodes-1)

   Curve.SampleCoordinates(sCurve, dSample, nodes(i,1), nodes(i,2), nodes(i,3))

   Debug.Print(Str$(dSample) + ": x=" + Str$(nodes(i,1)) + ", y=" + Str$(nodes(i,2)) + ", z=" + Str$(nodes(i,3)))

Next
