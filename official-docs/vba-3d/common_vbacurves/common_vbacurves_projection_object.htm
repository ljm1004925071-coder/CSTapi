Projection Object

This object enables a projection operation from a several number of curve items on a solid.

  • Methods

  • Default Settings

  • Example

Methods

Reset

Resets all internal settings to their default values.

 

Name ( name curvename )

Sets the name of the first curve to project.

 

AddName ( name curvename  )

Sets the name of the curve to project.

 

TargetSolid ( name solidname  )

Sets the name of the target solid on which to project the curves.

 

EmbedInTarget ( bool embed)

Specify whether the projected curves shall be imprinted in the target solid. The default is false.

 

DeleteCurve ( bool delete)

Specify whether the original curves shall be deleted. This option can only be set when the curves are embedded in the target solid.

 

Project

Projects the curves toward the target solid along its surface normals.

Example

With Projection

      .Reset

      .Name "curve1:line1"

      .AddName "curve1:line2"

      .TargetSolid "component2:solid1"

.EmbedInTarget "False"

.DeleteCurve "False"

.Project

End With
