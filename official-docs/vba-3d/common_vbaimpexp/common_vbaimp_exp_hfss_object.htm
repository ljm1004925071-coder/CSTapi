HFSS Object

Methods

Reset

Resets the import options to the default.

 

FilenameHFSS ( filename  filename )

Sets the name of the hfss/aedt file.

 

FilenameSAB ( filename  filename )

Sets the name of the sab file.

 

FilenameSM3 ( filename  filename )

Sets the name of the sm3 file.

 

Read

Starts the import of the selected files. For the HFSS import an aedt and sab file is needed. For older HFSS projects an sm3 file can be used instead of the sab file.  

 

ImportData ( filename  AedtOrHfssfilename,  filename  SabOrSm3filename )

This is a scripting command. Execution of this command will copy the necessary file to project and adds HFSS import command to the model history.

In order to import AEDT project specify the aedt file name and the sab file name. To import HFSS project specify the hfss file name and the sm3 file name.
