ColourMapPlot Object

This object allows controlling the visualization of color map plots.

  • General Methods

  • Query Methods

General Methods:

PlotView ( string type ) bool

This method corresponds to selecting a certain plot of the plot toolbar. The variable type specifies the type of view that should be plotted and can be one of the following strings: "real", "imaginary", "magnitude", "magnitudedb", "phase". If the data could be plotted with the given view type, the method returns true, otherwise false.

 

SetXAutoTick / SetYAutoTick ( bool on )

If set to true (default), the plot ticks of the horizontal / vertical axis are chosen automatically as round values. Otherwise the number of ticks is defined by the SetXTicks / SetYTicks settings.

 

SetXTicks / SetYTicks ( long ticks )

Specifies the number of ticks along the horizontal / vertical axis. This method automatically turns the SetXAutoTick / SetYAutoTick switch off. Please note that at least two ticks must be specified.

 

SetXAutoRange / SetYAutoRange ( bool on )

This method enables or disables auto-scaling of the horizontal / vertical axis. If on is True (default) the plot will always be auto-scaled to the maximum range. Otherwise the range is defined by the SetXRange / SetYRange settings.

 

SetXRange / SetYRange ( double min, double max )

Specifies the range to be plotted for the horizontal / vertical axis. This method automatically turns the SetXAutoRange / SetYAutoRange switch off.

 

SetClampAutoRange ( bool on )

This method enables or disables auto-scaling of the color ramp. If on is True (default) the color ramp is auto-scaled to the maximum data range. Otherwise the range is defined by the SetClampRange settings.

 

SetClampRange ( double min, double max )

Specifies the data range for the color ramp scale. This method automatically turns the SetClampAutoRange switch off.

 

SetClampLogarithmic ( bool on )

This option enables or disables logarithmic color scaling of the clamp range.

 

SetDrawGrid ( bool on )

This option specifies whether a grid is drawn on top of the data.

 

SetKeepAspectRatio ( bool on )

This option specifies whether the result data's aspect ratio of horizontal axis and vertical axis is kept for the plot or both axes of the plot are stretched to fill the view completely.

 

SetSkipValuesMode ( enum mode )

This option specifies which data values shall be skipped from visualization.

enum mode

Skip values

"clamp" (default)

don't skip any values

"lessorequal"

less or equal threshold

"greaterorequal"

greater or equal threshold

"outsiderange"

outside clamp range

"insiderange"

inside clamp range

 

SetSkipValuesThreshold ( double threshold )

This threshold is used only if the SetSkipValuesMode option is set to "lessorequal" or "greaterorequal".

 

ExportImage ( string filename, long width, long height )

Creates a bitmap file of the current plot with the specified size. The string filename is expected to be an absolute file path.

 

Reset

Reset all plot options to their default values.

Query Methods

Please note that a more efficient interface to access and manipulate 2D Colormap data is available via the Result2D object.

 

GetXMinData / GetXMaxData / GetYMinData / GetYMaxData double

Get the x and y ranges of the data.

 

GetXMinView / GetXMaxView / GetYMinView / GetYMaxView double

Get the x and y ranges of the current view.

 

GetDataMinValue / GetDataMaxValue double

Get the minimum and maximum values of the data.

 

GetViewMinValue / GetViewMaxValue double

Get the minimum and maximum values of the subset of the data which is currently displayed in the view.

 

GetDataNx / GetDataNy long

Get the number of data samples along the x and y axes.

 

GetDataValue ( long x, long y ) double

Get the value of the data with the given indices x and y. Valid indices are greater or equal 0 and lower than the number of data samples in the corresponding direction.
