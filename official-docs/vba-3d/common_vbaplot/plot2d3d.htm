Plot2D3D Object

This object allows to access different plot settings of the current 2D/3D Plot.

  • General

  • Scaling

  • Ticking

  • Special Clamping

  • Animation Control

  • Cross Section

  • Drawings

General

SetStructureTransparent ( bool flag )

Plots the structure transparent.

 

SetHideRepetitions ( bool flag )

Don't show any repetitions of models using any kind of symmetry, e.g. symmetry planes, rotational symmetry, unit cell, etc. Instead only show that partition of the simulation domain that has been solved by the solver. Surface plots of Hex mesh results will ignore this setting though. Field evaluation in post-processing (like "Evaluate field in arbitrary coordinates", "Evaluate Field on Curve/Face") and "Post-Processing -> Import/Export -> Plot Data (ASCII)" will return zero valued results outside the simulated domain, when Hide repetitions is activated. Except for I-Solver results and Surface Currents for hex based results.

 

ExportImage ( string filename, long width, long height )

Creates a bitmap file of the current plot with the specified size. The string filename is expected to be an absolute file path.

Scaling

SetScalingMode ( enum mode )

Sets the scaling mode.

enum mode

scaling

"linear" (default)

linear scaling

"log"

logarithmic color scaling

"decibel"

dB scaling mode

 

SetClampAutoRange ( bool on )

This method enables or disables auto-scaling of the color ramp. If on is True (default) the color ramp is auto-scaled to the maximum data range. Otherwise the range is defined by the SetClampRange settings.

 

SetClampRange ( double min, double max )

Specifies the data range for the color ramp scale. This method automatically turns the SetClampAutoRange switch off.

 

SetLogStrength ( double strength )

Sets the log strength if the log scaling mode is active. Values from >1.0 to 100000 are allowed.

 

SetdBRange ( double range )

Sets the dB range used in the dB scaling mode.

 

GetLogarithmicFactor double

Returns the logarithmic factor of selected result. This factor is used for dB scaling.

 

ResetScaling

Resets internal scaling related settings to default settings.

Ticking

SetTickMode ( enum mode )

This option specifies how to distribute ticks on the color ramp range.

enum mode

distribution

"auto" (default)

automatic, using round values

"fixedcount"

equidistant, using a given tick count (cf. SetNumberOfContourValues)

"fixedstep"

equidistant, using a given tick step size(cf. SetStepSize)

 

SetNumberOfContourValues ( long number )

Changes the number of contour values used in "fixedcount" mode. The number must be greater 2, numbers greater 99 are not recommended for normal use. A higher value results in a smoother coloring.

 

SetStepSize ( double stepsize )

Changes the contour value step size used in "fixedstep" mode. A smaller value results in a smoother coloring.

 

DrawContourLines ( bool switch )

Outline the contour values with black lines between the color steps. Not available in "auto" mode.

 

Banding ( bool switch )

Specify whether to use a banded or continuous color map. Not available in "auto" mode.

Special Clamping

SetClampMode ( enum mode )

This option specifies which data values shall be skipped from visualization.

enum mode

Skip values

"clamp" (default)

don't skip any values

"hideoutside"

outside clamp range

"hideinside"

inside clamp range

"fixedoutside"

outside clamp range and color them in a fixed color

"fixedinside"

inside clamp range and color them in a fixed color

 

SetClampColor ( double red, double green, double blue )

Set user-defined color of the clamp modes "fixedoutside" and "fixedinside". Each value needs to be in the range [0.0, 1.0].

Animation Control

GetDomain enum

Returns the domain of the monitor used, return code can have the following values:

"frequency"

The plot is based on recordings of a frequency domain monitor.

"time"

The plot is based on recordings of a time domain monitor.

"static"

The plot is based on recordings of a static monitor.

 

GetNumberOfSamples long

Returns the total number of samples if the plot is based on recordings of a time monitor else 0.

 

GetTStart double

Returns the start time of recording if the plot is based on recordings of a time monitor else 0.0.

 

GetTEnd double

Returns the end time of recording if the plot is based on recordings of a time monitor else 0.0.

 

GetTStep double

Returns the time step width of recording if the plot is based on recordings of a time monitor else 0.0.

 

SetTime ( double time )

GetTime double

Specifies / returns the current  time set of a time monitor plot.

 

SetSample ( long sample )

GetSample long

Specifies / returns the current time sample number of a time monitor plot.

 

PhaseValue ( double phase )

Specifies the phase of to be plotted complex field values.

 

PhaseStep ( double step )

Specifies the step width used by phase increment action.

Cross Section

DefineCrossSection ( enum {0, 1, 2, 3} id, double nx, double ny, double nz, double px, double py, double pz )

Defines position and orientation for one of four available Cross Sections by the normal direction and point on the plane.

 

ActivateCrossSection ( enum {0, 1, 2, 3} id, bool switch )

Activates or deactivates one of four available Cross Sections. After activation, the given Cross Section will automatically be selected for manipulation.

 

SelectCrossSection ( enum {0, 1, 2, 3} id )

Selects one of four available Cross Sections for manipulation.

 

DefineCutplane ( double nx, double ny, double nz, double px, double py, double pz )

Defines the cutplane by the normal direction and point on the plane.

 

ActivateCutplane ( bool switch )

A cutplane can be defined to look inside a structure. This cutplane divides the structure into two parts, while only one of them will be plotted.

Drawings

AddLines ( variant dArray )

Add line drawing to the 2D/3D result plot.

The array needs to start with index zero and is expected to contain a series of 3D points consisting of x,y,z-coordinates, where each pair of points defines a line segment (e.g. x0,y0,z0, x1,y1,z1, x2,y2,z2, x3,y3,z3 will produce two line segments p0-p1 and p2-p3).

 

DeleteLines

Remove line drawing from the 2D/3D result plot.

 

AddPoints ( variant dArray )

Add marker points to the 2D/3D result plot.

The array needs to start with index zero and is expected to contain a series of 3D points consisting of x,y,z-coordinates, where each point defines a marker position.

 

DeletePoints

Remove marker points from the 2D/3D result plot.

 

SetDrawColor ( double red, double green, double blue )

Set color of line drawing and marker points. Each value needs to be in the range [0.0, 1.0].

 

SetDrawThickness ( long thickness )

Set thickness of line drawing and marker points.
