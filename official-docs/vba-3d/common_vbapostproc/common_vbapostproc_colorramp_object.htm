ColorRamp Object

Use this to change the color order for the color-by-value field plots.

  • General Methods

  • Deprecated Methods

General Methods

Type ( enum name )

Sets the color ramp to a pre-defined or user-defined color order.

enum name

color order

"Rainbow"

blue-cyan-green-yellow-red (default)

"Fire"

cyan-blue-magenta-red-yellow

"Inspire"

green-blue-magenta-red-yellow

"FarFire"

blue-magenta-red-yellow

"Hot"

black-red-yellow-white

"Gray"

for black-and-white printer

"Phase"

red-yellow-green-cyan-blue-magenta-red

"Free"

user-defined color order

"Mono"

white to user-defined color

 

AddFreeRampVertex ( single value, single red, single green, single blue )

Adds a vertex to the user-defined "Free" color ramp. All parameters need to be in the range [0.0, 1.0]. The value defines the relative position on the color ramp, with value = 0.0 referring to the lower bound of the color ramp and value = 1.0 to the upper bound. The color at this position of the color ramp is given in red, green, blue.

 

DeleteFreeRampVertices

Delete all vertices from the user-defined "Free" color ramp. This results in an empty black color ramp.

 

SetFreeRampMode ( enum name )

The "Free" color ramp can operate in one of two modes: "continuous" (default) or "stepped"

In "continuous" mode the colors are interpolated linearly between the given color ramp vertices.

For example a simple color ramp ranging from blue to red in a continuous transition is defined as follows:

SetFreeRampMode ("continuous")

AddFreeRampVertex (0.0, 0.0, 0.0, 1.0)

AddFreeRampVertex (1.0, 1.0, 0.0, 0.0)

In "stepped" mode each color covers the range from the given color ramp vertex up to the next color ramp vertex.

For example a simple color ramp with blue in the lower half and red in the upper half is defined as follows:

SetFreeRampMode ("stepped")

AddFreeRampVertex (0.0, 0.0, 0.0, 1.0)

AddFreeRampVertex (0.5, 1.0, 0.0, 0.0)

 

SetMonoRampColor ( single red, single green, single blue )

Set user-defined color of the "Mono" color ramp. Each value needs to be in the range [0.0, 1.0]. This results in a color ramp ranging from white to the given color.

 

Invert ( bool flag )

Inverts the color order, e.g. turn the rainbow type to red-yellow-green-cyan-blue.

 

Style ( enum name )

This switch either hides ("None") the color ramp or positions it vertically ("Vertical") in the main view.

 

Reset

Resets internal settings to default settings.

Deprecated Methods

Methods related to 2D/3D Plots only have been relocated to Plot2D3D object. Methods related to Farfield Plots only have been relocated to FarfieldPlot object.

Thus, the following methods are marked deprecated and will vanish from the interface in the future. Please consider updating your scripts accordingly.

 

SetScalingMode ( enum mode )

SetClampRange ( double min, double max )

SetLogStrength ( double strength )

SetdBRange ( double range )

Please refer to Plot2D3D object for descriptions.

 

NumberOfContourValues ( long number )

GetNumberOfContourValues long

DrawContourLines ( bool switch )

Please refer to Plot2D3D object or FarfieldPlot object for descriptions.
