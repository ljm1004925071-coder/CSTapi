FaceDistortion Object

This object is used to create face distortions on sheets.

  • General Methods

  • Example

General Methods

Reset

Resets all internal settings to their default values.

 

Name ( solidname SolidName )

Sets the name of the sheet to distort.

 

NumberDistortionsInX (  int x )

Define the number of distortions in x direction. Change this value to form different patterns on a surface. The distance between the distortions is constant.

 

NumberDistortionsInY (  int y )

Define the number of distortions in y direction. Change this value to form different patterns on a surface. The distance between the distortions is constant.

 

MinimunDistortionHeight (  int min )

Define the minimum height of the distortions i.e. the lower boundary of the peak-to-peak interval. The distribution of the height values is uniform.

 

MaximumDistortionHeight (  int max )

Define the maximum height of the distortions i.e. the upper boundary of the peak-to-peak interval. The distribution of the height values is uniform.

 

DistortionPatternId (  int id)

Define the distortion pattern id to reliably reproduce the distortions at any time. The distortion id is used in the algorithm as a seed for the generation of pseudo-random integer numbers. The distortions are uniformly over the parameter range of the surface. The distortion high starts always with the same random value which is initialized via the distortion pattern. 

 

Create

Create the distortions onto the face.

Example

With FaceDistortion

     .Reset

     .Name "component1:sheet"

     .NumberDistortionsInX "20"

     .NumberDistortionsInY "30"

     .MinimunDistortionHeight "-0.5"

     .MaximumDistortionHeight "0.5"

     .DistortionPatternId "1"

     .Create

End With
