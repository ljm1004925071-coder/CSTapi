Methods for Units

SetUnit (enum dimension, enum unit)

Sets the entered unit for the selected dimension. Admissible values for dimension and unit are:

dimension

supported units

Length

nm, um, mm, cm, m, mil, in, ft

Time

fs, ps, ns, us, ms, s

Frequency

Hz, kHz, MHz, GHz, THz, PHz

Temperature

degC, K, degF

Unit strings are case-sensitive and follow the syntax described in the units documentation.

 

GetUnit (enum dimension)

Returns the current unit setting for dimension. See SetUnit for more information on the returned unit strings.

Methods for Geometric Units

GetGeometryUnitToSI double

Returns the factor to convert a geometry value measured in units of the current project into the units.

-> a(SI Unit) = factor * b(project Unit)

 

GetGeometrySIToUnit double

Returns the factor to convert a geometry value in SI units into the units defined in the current project.

-> a(project Unit) = factor * b(SI Unit)
