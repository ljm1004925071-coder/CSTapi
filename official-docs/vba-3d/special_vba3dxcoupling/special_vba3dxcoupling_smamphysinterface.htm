SMAMPhysInterface Object

Defines a 3DEXPERIENCE multi-physics interface to exchange solution results with other SIMULIA solvers.

  • Methods

  • Examples

Methods

Reset

Resets all internal values to their default settings.

 

PortID ( string portID )

Provides the unique ID for the port. The string must contain a hexahedral value.

 

PortName ( name portName )

Provides the name of the port which will be visible in 3DEXPERIENCE infrastructure. Must be non-empty and unique.

 

PortType ( enum portType )

Provides type enum for the port. Currently only "OutLFForceMonitor" is supported.

 

OutLFForceMonitor

Output multi-physics port containing the results of an LF Force and Torque monitor.

 

ResultType ( enum resultType )

Provides type enum for the result.

 

Signal

The port delivers a 0D or 1D-result.

Distribution

The port delivers a 2D or 3D field distribution.

 

AddValueType ( enum valueType )

Adds a field value type to the list for the specific port.

 

Force

The port delivers Force values in N.

Torque

The port deliversTorque values in N*m.

 

 

AddFrequency ( double frequency )

Adds a frequency value to the list for the specific port.

 

MonitorName ( name monitorName )

If a port is supposed to share data for a specific monitor, provide here the monitor name. Only LF Force and Torque monitors are supported.

 

CreatePort

Creates a new port with the given settings. This command fails if a port with the same name exists.

 

ModifyPort

Modifies settings for a port with the give name. This command fails if a port with this name does not exist.

 

RenamePort ( name oldName, name newName )

Renames a port with the name oldName to newName. The command fails if no port with the name oldName exists or a port with the name newName already exists.

 

ChangeMonitorNameForPortType ( enum portType, name oldName, name newName )

For all the ports of type portType with the monitor name oldName assigned, changes the assigned monitor name to newName. This command is helpful if a monitor has been renamed.

 

DeletePort ( name portName )

Deletes the port with the name portName.

 

DeleteAllPorts

Deletes all the 3DEXPERIENCE multi-physics interface ports.

Examples

Create a new 3DEXPERIENCE multi-physics interface port for a Force and Torque monitor:

 

With SMAMPhysInterface

    .Reset

    .PortName "Force on magnet2"

    .PortID "302d6ad1e92c4474934cbc2bc78c7ef7"

    .PortType "OutLFForceMonitor"

    .ResultType "Signal"

    .AddValueType "Force"

    .AddValueType "Torque"

    .MonitorName "Force on magnet2"

    .CreatePort

End With
