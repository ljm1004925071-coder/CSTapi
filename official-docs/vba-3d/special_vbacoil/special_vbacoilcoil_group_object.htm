Coil Group Object

Defines a new goup of coils or coil segments.

  • Methods

  • Default Settings

  • Examples

Methods

Reset

Resets the default values.

 

Name ( name  name )

Sets the name of the new coil group.

 

Type ( enum {"Current", "Voltage"} type )

Specifies the type of the coil group, which can be "Current" or "Voltage".

 

Value ( double  value)

A value of current or voltage depending on the coil group type.

 

Phase ( double  phase )

Sets the group phase  in degrees. This setting affects only the simulations in  frequency domain.

 

Color ( double  r, double  g, double  b)

Sets the color of the group using an RGB color value representation.

 

Create

Creates coil group with previously defined settings.

 

Change

Updates existing group.

 

Import

Used in sub-projects to import coil groups from master project.

 

Delete  ( name  name )

Deletes specified coil group.

 

Rename ( name  oldname,  name  newname )

Renames an existing coil group.

 

AddCoil (name  name )

Adds coil to an existing coil group.

 

RemoveCoil (name  name )

Removes coil from an existing coil group.

 

GetNumberOfCoilGroups ()

Returns a number of coil groups.

 

GetGroupNameFromIndex ( int  index)

Returns a coil group name for the specified coil index or an empty string, if the coil item is not grouped.

 

GetGroupNameForItem ( name  name )

Returns a coil group name for the specified coil item or an empty string, if the coil item is not grouped.

 

GetTypeFor Group (name  name)

Returns coil group type for a specified group.

 

GetValueForGroup (name  name)

Returns calculated source value for a specified group.

 

GetPhaseForGroup (name  name)

Returns calculated phase value for a specified group.

 

DoesGroupExist (name  name)

Checks whether a coil group exists.

Default Settings

Type ("Current")

Value ("0.0")

Phase ("0.0")

Example

With CoilGroup

     .Reset

     .Name "CoilGroup"

     .Type "Current"

     .Value "10"

     .Phase "30"

     .Color "0.4980 ", "0.7450", "1.0"

     .Create

End With
