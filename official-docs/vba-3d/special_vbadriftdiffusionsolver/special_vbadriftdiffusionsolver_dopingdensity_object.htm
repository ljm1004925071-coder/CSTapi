Doping Density Object

Defines a new impurity on a solid.

Methods

Reset

Resets all internal settings to their default values.

 

Name (name  name)

Sets the name of the new impurity.

 

Folder (name  foldername)

Sets a name of a folder. This helps either separating different objects within the navigation tree. The block name is automatically added during a sub-project import.

 

Enable (bool  activity)

Enables the source for the simulation.

 

DensityType (name  doping)

Sets the doping type to either "Acceptor" or "Donator".

 

Distribution (name  method)

Sets the method for the doping density distribution within the selected solids to "constant" or "Import".

 

Density (double  value)

Sets the doping density value within a solid, which is specified by the face command. The value is inserted in 1/cm^3. This option is available, if the "Import" method is selected.

 

FieldImport (name  field_source)

Uses the imported field source for the doping density distribution.

 

AddFace (solidname  solidname, int  faceid)

Selects a face from a solid by its face id, where the source is mapped to.

 

Create

Creates the source with the previously made settings.

 

Delete (name  name)

Deletes the specified doping density source.

 

Rename (name  oldname, name  newname)

Renames the specified doping density.

 

GetValue (name name)

Returns the combined term of acceptors (positive) and donators (negative) density.

 

Default Settings

Name "dopingdensity1"

Folder ""

Enable "True"

Acceptor "0"

Donator "0"

Example

With DopingDensity

     .Reset

     .Name "dopingdensity1"

     .Folder ""

     .Enable "True"

     .Acceptor "0"

     .Donator "0"

     .AddFace "silicon:solid1", "6"

     .Create

End With
