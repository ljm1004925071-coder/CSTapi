Doping Density Field Import

Imports a new 3D doping density field into CST Studio.

Methods

Reset

Resets all internal settings to their default values.

 

Name (name  name )

Sets the name of the new field import.

 

FileName (name  filename )

Sets the file name including the path, which will be imported.

 

ScalingFactor (double  rate)

Sets the scaling factor for the imported field. This factor is applied within the drift-diffusion solver and not during import.

 

Create

Creates a doping density field import item without importing it.

 

Delete (name  name)

Deletes the specified doping density field import..

 

Rename (name  oldname, name  newname )

Renames the specified doping density field import.

 

Default Settings

Name "dopingdensity_import1"

FileName ""

ScalingFactor "1.0"

Example

With DopingDensityFieldImport

     .Reset

     .Name "dopingdensity"

     .FileName "C:/data/DopingDensityImport.dat"

     .ScalingFactor "1.0"

     .Create

End With
