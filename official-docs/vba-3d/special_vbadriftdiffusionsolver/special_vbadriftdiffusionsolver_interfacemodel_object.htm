Interface Model Object

Defines a new interface model.

Methods

Reset

Resets all internal settings to their default values.

 

Name (name  name)

Sets the name of the new semiconductor-insulator interface model.

 

Folder (name  foldername)

Sets the name of a folder. This helps separating different objects within the navigation tree. The block name is automatically added during a sub-project import.

 

Enable (bool  activity)

Enables the source for the simulation.

 

InterfaceCharge (double charge)

Sets the charge at the interface. This terms modifies the potential.

 

Traplevel (double level)

Sets the energy level of the interface trap with respect to the middle of the band-gap.

 

SurfaceRecombinationModel (string  Species , double  SurfaceRecombinationVelocity, double  Nref)

Sets the parameter of the surface recombination model. Available species are "electron" and "hole".

 

AddFace (solidname  solidname, int  faceid)

Selects a face from a solid by its face id, where the source is mapped to.

 

Create

Creates the source with the previously made settings.

 

Delete (name  name)

Deletes the specified interface source.

 

Rename (name  oldname, name  newname)

Renames the specified interface model.

 

Default Settings

Name "Interface Model 1"

Folder ""

Enable "True"

InterfaceCharge "0."

TrapLevel "0."

SurfaceRecombinationModel "Electron", "1.e5", "0."

SurfaceRecombinationModel "Hole", "1.e5", "0."

AddFace "silicon:solid1", "1"

Example

With InterfaceModel

     .Reset

     .Name "Interface Model 1"

     .Folder ""

     .Enable "True"

     .InterfaceCharge "0."

     .TrapLevel "0."

     .SurfaceRecombinationModel "Electron", "1.e5", "0."

     .SurfaceRecombinationModel "Hole", "1.e5", "0."

     .AddFace "silicon:solid1", "1"

     .Create

End With
