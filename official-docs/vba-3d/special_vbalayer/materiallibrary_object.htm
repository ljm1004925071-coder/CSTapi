MaterialLibrary Object

The Material library represents a database where materials can be stored and restored. The MaterialLibrary Object lets you import or export material from or to the Material library.

  • Methods

  • Examples

Methods

 

LoadMaterialFromLibrary ( name matName, name folderName,  bool Replace ) bool

Import the material 'matName' from the Material library. If ‘folderName’ is empty material will be added to the Root folder. If option ‘Replace’ is True existing material in the Modeler will be replaced. This Method returns True if Material is loaded from the Material library.

 

UpdateMaterialPropertiesFromLibrary ( name matToBeUpdated, name matFromLibrary ) bool

Updates the Modeler material  properties from the Material library material. If successful Modeler material ' matToBeUpdated' will have properties of  'matFromLibrary' material. This Method returns True if Material properties are updated from the Material library.

 

UpdateAllMaterialsFromLibrary (  ) bool

Use this method to update properties of all the materials present in the Modeler. This Method returns True if Material properties are updated from the Material library.

 

SaveMaterialToLibrary ( name matName,  bool Replace ) bool

Saves the Modeler material 'matName' to the Material library. If option ‘Replace’ is True existing material in the  Material library will be overwritten. This Method returns True if Material is saved to the Material library.

 

GetAllMaterialsContainingString ( name substr) name

Get all the material names having 'substr' in their name from the material library. This method returns a comma separated string containing all the filtered material names.

 

GetMaterialListFromLibrary( ) name

           This method returns  a comma separated string containing  all the available material names from the Material library.

Examples

With MaterialLibrary

 

' Aluminum material will be imported to 'test' folder

     .LoadMaterialFromLibrary ("Aluminum","test",True)

 

End With

 

With MaterialLibrary

 

' Aluminum material present in 'test' folder will be updated using 'Copper (pure)' material from the material library

     .UpdateMaterialPropertiesFromLibrary ("test/Aluminum", "Copper (pure)")

 

End With
