Component Object

The Component Object lets you define or change components. Each solid is sorted into a component.

  • Methods

  • Example

Methods

New ( name componentname )

Creates a new component with the given name.

 

Delete ( name componentname )

Deletes an existing component and all the containing solids.

 

DeleteAllEmptyComponents ( name componentname )

Deletes an empty component and all empty sub components.

 

Rename ( name oldname, name newname )

Changes the name of an existing component.

 

DoesExist (name componentname ) bool

Returns whether a component with the given name does exist.

 

GetNextFreeName name

Returns the next unused component name.

 

Hiding and showing is generally not put into the project's history. Still, you can use these VBA commands to toggle visibility of dedicated objects.

 

HideComponent( name componentname )

ShowComponent( name componentname )

Hides or shows all shapes  within a component and its sub components.

 

Hide()

Show()

Hides or shows the currently selected objects.

 

HideUnselected()

ShowUnselected()

Hides or shows the currently not  selected objects.

 

HideAll()

ShowAll()

Hides or shows all hideable objects.

 

HideAllPorts()

ShowAllPorts()

Hides or shows all port objects.

 

HideAllFieldSources()

ShowAllFieldSources()

Hides or shows all field source objects.

 

HideAllLumpedElements()

ShowAllLumpedElements()

Hides or shows all lumped elements.

 

HideAllWires()

ShowAllWires()

Hides or shows all wires.

 

HideAllDielectric()

ShowAllDielectric()

HideAllMetals()

ShowAllMetals()

Hides or shows all shapes of the described kind of material.

Example

' Create a new component

Component.New "component1"

 

' Rename an existing component

Component.Rename "component1", "MyComponent"

 

' Hiding and showing different kind of objects.

Component.HideComponent "antenna/feed"

Component.Show
