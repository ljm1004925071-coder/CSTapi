MeshShapes Object

This object controls the mesh generation. Several methods/functions to control the automatic mesh generation as well as to define a manual mesh can be found here.

In both cases the major element to control the mesh generator are fixpoints. These fixpoints force the mesh generator to try to position mesh lines through this point. However, it might not always be able to do so, because of concurring settings.

Fixpoints can be accessed via a unique identity number.

Methods

Reset

Resets all internal settings to their default values.

 

Delete ( string MeshShapeName )

Deletes the specified mesh shape.

 

Rename ( string OldMeshShapeName, string NewMeshShapeName )

Renames the existing mesh shape to the new name.

 

NewFolder ( name FolderName )

Creates a folder for mesh shapes with the specified name.

 

DeleteFolder ( name FolderName )

Deletes the folder and all containing mesh shapes.

 

RenameFolder ( string OldFolderName, string NewFolderName )

Renames the existing mesh shape folder to the new name.

 

ChangeMaterial ( string MeshShapeName, name MaterialName )

Changes the material of a mesh shape. The material to change to must already exist.

 

AddName ( string ElementName )

Adds an element name to an internal list, which will be used by functions which are working with multiple elements.
Mesh shape names can be added if afterwards the functions DeleteMultiple or ResolveIntersections are invoked.
Solid names can be added if afterwards the functions CreateMeshShapesByFacetting or CreateMeshShapesByRemeshing are invoked, the latter allows both solid and mesh shape names.

 

DeleteMultiple

Deletes multiple mesh shapes at once. The mesh shapes must be added  with the command AddName. If several mesh shapes should be deleted, the performance will be much better if this function is used, instead of using function Delete multiple times.

 

Tolerance ( double value )

Specifies the tolerance which will be used in function ResolveIntersections.

 

ResolveIntersections

Repaires self-intersections between mesh shapes. The mesh shapes must be added  with the command AddName.

 

MeshElementSize ( double value )

Specifies the targeted size of mesh elements, used by functions CreateMeshShapesByFacetting and CreateMeshShapesByRemeshing.

 

CreateMeshShapesByFacetting

Convert each solid to a mesh shape, while retaining all geometric detail of the solid. The solids must be added  with the command AddName. The specified mesh element size controls the desired resolution of the mesh. However, to faithfully represent small geometric details and strongly curved surfaces, the mesh elements may become smaller. If multiple intersecting solids were selected, then the generated meshes will also intersect. Shape names and material assignments are preserved.

 

CreateMeshShapesByRemeshing

Creates a mesh that approximates the given solids and mesh shapes. The solids and mesh shapes must be added  with the command AddName. The selected mesh element size specifies the resolution for the approximation. The remeshing operation resolves intersections, can remove small geometric details and close small gaps. Names and material assignments of the selected shapes are lost during this step.

Default Settings

The given values refer to High Frequency simulations and might differ for Low Frequency setup.

Tolerance (1e-6)

MeshElementSize (0.1)

Example

With MeshShapes

     .Reset

     .Tolerance "1e-6"

     .AddName "folder1:shape1"

     .AddName "folder1:shape2"

     .ResolveIntersections

End With
