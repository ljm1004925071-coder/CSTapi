PICPhaseSpaceMonitor Object

Defines a phase space monitor for the PIC solver.

  • Methods

  • Default Settings

  • Examples

Methods

Reset

Resets all internal values to their default settings.

 

Name ( name monitorName )

Sets the name of the monitor.

 

XType ( enum dir )

This method defines the type of the abscissa of the phase space plot. Possible values are:

  • Position

  • Normed Momentum

  • Gamma

  • Beta

  • Velocity

  • Energy

XComponent ( enum dir )

This method defines the component of the abscissa of the phase space plot. Possible values are X, Y, Z and Abs.

 

YType ( enum dir )

This method defines the type of the ordinate of the phase space plot. Possible values are:

  • Position

  • Normed Momentum

  • Gamma

  • Beta

  • Velocity

  • Energy

YComponent ( enum dir )

This method defines the component of the ordinate of the phase space plot. Possible values are X, Y, Z and Abs.

 

Tstart ( double startTime )

Sets starting time for a time domain monitor to startTime.

 

Tstep ( double timeStep )

Sets the time increment for a time domain monitor to timeStep.

 

Tend ( double stopTime )

Sets the end time for a time domain monitor to stopTime.

 

UseTend ( bool bFlag )

If bFlag is TRUE, the time domain monitor will stop storing the results when the end time set with Tend is reached. Otherwise the monitor will continue until the simulation stops.

 

FType ( name name )

Select a particle source for filtering during runtime. Only particles emitted by this source are considered. The default option none considers all particles.

 

Delete ( name monitorName )

Deletes the monitor named monitorName.

 

Rename ( name oldname,  name newname )

Renames the monitor named oldname to newname.

 

Create

Creates the monitor with the previously applied settings.

Default Settings

Tstart (0.0)

Tstep  (0.0)

Tend   (0.0)

UseTend (FALSE)

Examples

' creates a PIC Phase Space Monitor

 

With PICPhaseSpaceMonitor

.Reset

.Name "pic phase space monitor 1"

.XType "Position"

.XComponent "Z"

.YType "Normed Momentum"

.YComponent "Z"

.Tstart "0.0"

.Tstep "0"

.Tend "0.0"

.UseTend "False"

.FType "none"

.Create

End With
