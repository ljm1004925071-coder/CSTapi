VBA Objects Overview

CST Studio Suite is implemented as OLE automation server. The application is registered under "CSTStudio.Application". An instance can be created through the CreateObject  VBA command:

Dim app As Object

Set app = CreateObject ("CSTStudio.Application")

This application object provides methods to open and create projects. Refer to the Application Object  for details.

If you open the built-in CST Studio Suite VBA-Interpreter all of these objects are already available. To call the method AMethod of the object AnObject simply type AnObject.AMethod ( Parameters ). All functions and methods of the Project Object and the Units Object can be called directly, because you are in the scope of this object.

The VBA commands for the associated canvas can be accessed as well. Refer to the CST Design Studio objects page that gives a short overview over their usage.
