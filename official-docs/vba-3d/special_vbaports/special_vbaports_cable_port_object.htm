CablePort Object

Use this object to define cable ports. Beside waveguide ports or plane waves the cable ports offers another possibility to feed the calculation domain with power.

Different types of cable ports are available, considering the excitation as a current source, voltage source or as an impedance element which also absorb some power and enables S-parameter calculation.

Methods

Reset

Resets all internal values to their default settings.

 

Create

Creates a new cable port. All necessary settings have to be made previously.

 

Modify

Modifies an existing cable port. Only none geometrical settings which were made previously are changed.

 

AddTerminalPort ( string port name, string port type, string port value, bool monitor, string node1 , string terminal1, string node2, string terminal2, string sub folder, string port label )

Provides the parameters to create the wire terminal type cable port.

 

port name: String containing the unique name for the new port

port type: One of 'sparameter', 'voltage' or 'current'

port value: the corresponding value for the port type, sparameter (50 Ohm default), voltage (1V default) or current (1A default)

monitor: Boolean value to control whether to create a monitor at this port, one of true or false

connector1, connector2: String containing the id of the nodes, node2 is optional, used only for differential ports

terminal1, terminal2: String containing the id of the wire terminals, terminal2 is optional, used only for differential ports

sub folder: String containing the name of the folder in the navigation tree the port to be is located in

port label: A unique label for the port in the result folder

 

AddPinPort ( string port name, string port type, string port value, bool monitor, string connector1 , string plugin1 , string pin1, string connector2, string plugin2, string pin2, string sub folder, string port label )

Provides the parameters to create the connector pin type cable port.

 

port name: String containing the unique name for the new port

port type: One of sparameter, voltage or current

port value: String containing the corresponding value for the port type, sparameter (50 Ohm default), voltage (1V default) or current (1A default)

monitor: Boolean value to control whether to create a monitor at this port, one of true or false.

connector1, connector2: String containing the id of the connectors, connector2 is optional, used only for differential ports

plugin1, plugin2: String containing the id of the connector plugins, plugin2 is optional, used only for differential ports

pin1, pin2: String containing the id of the connector pins, pin2 is optional, used only for differential ports

sub folder: String containing the name of the folder in the navigation tree the port to be is located in

port label: A unique label for the port in the result folder

 

.AddStringArgument(string commandId, string parameter)

commandId: one of 'name' (with the port name as parameter) and one of 'param' (with one of the commands below as parameter) to query the given param of the given port.

parameter: one of the following options.

 

.GetString("QueryPortProperty")

Queries the following properties of the cable port specified by commandId 'name' and parameter 'portnumber'.

CommandId param with parameter 'Type' returns one of the following strings: "sparameter", "voltage" or "current".

CommandId param with parameter 'Label' will return the label of the port.

 

.GetLong("QueryPortProperty")

Queries the following property of the cable port specified by commandId 'name' and parameter 'portnumber'.

CommandId param with parameter 'id' returns the id of the port.

 

.GetBool("QueryPortProperty")

Queries the following property of the cable port specified by commandId 'name' and parameter 'portnumber'.

CommandId param with parameter 'is_differential' returns one of the strings "True" or "False".

 

.SelectPort(string commandId)

Selects the cable port specified by commandId.

CommandId param specifies the index of the port, e.g. port 1 is '1'.

 

.SetData(string type, string value, string addMonitor)

Sets data for the selected port (using SelectPort).

type is one of current or 'sparameter', 'voltage' or 'current'

value is a numeric value for Ohm (type sparameter), Volt (type voltage) or Ampere (type current)

Parameter addMonitor is one of 'True' or 'False'. It specifies whether to add a monitor as well.

 

.Invert(string value)

Inverts the direction of the currently selected port. Parameter value is one of 'True' or 'False'.

Default Settings

Port name (1)

Label ("")

Type ("SParameter")

Impedance (50.0)

Voltage (1.0)

Current (1.0)

Monitor (False)

 

Example

' Define a cable port using a terminal

With CablePort

     .Reset

     .AddTerminalPort("42", "sparameter", "51", "true", "N1", "N1_SW_1", "N5", "N5_RC_1_1", "MyFolder", "MyLabel")

     .Create

End With

 

' Define a cable port using connector pins

With CablePort

     .Reset

     .AddPinPort("13", "current", "1.2", "false", "Connector", "Plugin_1", "Pin_1", "Connector", "Plugin_1", "Pin_2", "MyFolder", "MyOtherLabel")

     .Create

End With

 

'Delete cable ports

With CablePort

    .Reset

    .SelectPort 13

    .SelectPort 42

    .Delete

End With
