ScatterResult Object

This object offers access and manipulation functions for scatter results. This object is available in 3D Simulation and Circuits & Systems.

A ScatterResult object represents an arbitrary number of discrete data points in 3D space. Each data point can hold an arbitrary number of data  values representing a physical quantity. The data values can consist of scalar or vector values with real or complex components.

Initialization, File Operation

Construction

A ScatterResult object can be created as follows:

 

Dim result As Object

Set result = ScatterResult("")

 

This will create an empty object. Alternatively, a file name can be given as a parameter, then the object loads the data from the file.

 

Initialize ( long nSize )

Initializes an empty ScatterResult object with the specified number of 3D data points.

 

Load ( string sFileName )

Loads the ScatterResult object from file.

Note: The names used in the ResultTree do not necessarily correspond to the file names. The file names of tree items can be queried using the ResultTree Object.

 

Save ( string sFileName )

Saves the ScatterResult object to file.

 

AddToTree ( name sTreePath )

Inserts the ScatterResult object into the tree at the folder specified by sTreePath.

Note: Save the ScatterResult object before adding it to the tree to set correct path.

General Settings

SetTitle ( string sTitle )

GetTitle string

Sets / returns the title of the result.

 

AddQuantity ( string sName, bool bScalar, bool bComplex )

Adds a new quantity to the result. The quantity can be accessed with the given unique name and will contain data values of the given properties (scalar or vector and real or complex).

 

GetQuantityNames variant

Returns a list of names that can be used to access the quantities that are stored in the result.

 

SetUnit ( string sName, string sUnit )

GetUnit ( string sName ) string

Sets / returns the unit of the quantity with the given name as string. The unit string uses the syntax described on the Units help page.

 

SetLogarithmicFactor ( string sName, double dValue )

GetLogarithmicFactor ( string sName ) double

Sets / returns logarithmic factor used for dB scaling of the quantity with the given name. The logarithmic factor can be 10 for power quantities, 20 for field quantities and zero if dB scaling is not available.

Data Access

GetSize long

Returns the number of 3D data points.

 

IsScalar ( string sName ) bool

Returns true if the quantity with the given name holds scalar data and false if it holds vector data.

 

IsComplex ( string sName ) bool

Returns true if the quantity with the given name holds complex-valued data.

 

SetPositions ( variant dArray )

GetPositions variant

Sets / returns 3D positions of  all data points stored in the ScatterResult object from / as a double array.

The array needs to start with index zero and is expected to have 3 times as many elements as the ScatterResult object's number of data points. Consider using the method 'GetSize' to query and 'Initialize' to modify the number of data points of the ScatterResult object. The array's data layout is x1,y1,z1, x2,y2,z2, ..., xN,yN,zN.

 

SetValues ( string sName,  variant dArray )

GetValues ( string sName ) variant

Sets / returns data values of the quantity with the given name of all data points stored in the ScatterResult object from / as a double array.

Example

This example creates an empty object, fills it with real-valued data and adds it to the 3D Simulation result tree.

 

Dim result As Object

Set result = ScatterResult("")

 

Dim positions() As Double

ReDim positions(4*3 - 1) ' 4 times x,y,z

' set positions array data here

 

Dim values() As Double

ReDim values(4*2 - 1) ' 4 times re,im

' set values array data here

 

With result

  .Initialize(4)

  .SetPositions(positions)

  .AddQuantity("F-Parameter", True, True)

  .SetValues("F-Parameter", values)

  .SetTitle("My Scatter Result")

  .Save("my_result.rex")

  .AddToTree("2D/3D Results\Test\My Scatter Result")

End With
