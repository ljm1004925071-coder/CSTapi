Arfilter Object

Defines the settings for signal prediction. Signal prediction may be applied to time signals at the ports, probe signals or current and voltage monitor signals. You may define different settings for port, probe and monitor signals. A Signal prediction run can be started by using the PostProcess1D object.

Methods

SetType ( enum{"s-parameter", "probe", "monitor"} type )

Sets the type of signals the signal prediction settings are applied to.

 

SetAlgorithmType( enum{"Basic", "Enhanced"} type )

Sets the type of algorithm for the signal prediction.

The behavior of the "Basic (Default)" algorithm can be configured via the five options listed below. The "Enhanced (Preview)" version does not require any configuration.

 

SetFirstTime (string time )

The first time step (in seconds) from which filter determination starts. Instead of a numerical value you can also use "auto" for the setting to be determined automatically.

 

SetSkip ( long skip )

The number of time steps which will be skipped after an AR-filter has been found and before the next filter will be determined.

 

SetMaxFrq ( string fMax )

Because the time signals will be low pass filtered within the AR calculation process, the maximum frequency of the low pass filter can be specified here.  Instead of a numerical value you can also use "auto" for the setting to be determined automatically.

 

SetMaxOrder ( long order )

Sets the number of recursive filter elements. Higher order filters usually deliver more accurate results, but the calculation time increases parabolically with the number of filter elements (max. 100 elements).

 

SetWindowLength ( double length )

This method sets the length of the analyzed time interval relative to the maximum number of elements. E.g. with maximum order of filter 40 and a window length of 2.0 the number of analyzed time samples is 40 * 2.0 = 80.

Default Settings

SetAlgorithmType ("Basic")

SetType ("s-parameter")

SetFirstTime ("auto")

SetSkip (10)

SetMaxFrq ("auto")

SetMaxOrder (40)

SetWindowLength (2.0)

Example

With Arfilter

    .SetType ("s-parameter")

    .SetFirstTime (0.1777274577508)

    .SetSkip (10)

    .SetMaxFrq ("fMax * 1.2")

    .SetMaxOrder (40)

    .SetWindowLength (2.0)  

End With
