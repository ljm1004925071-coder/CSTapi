HeatPipe Object

This object is used to specify a heat pipe for the Conjugate Heat Transfer Solver.

  • Methods

  • Example

Methods

Reset

Resets all internal settings of the heat pipe to their default values.

 

Name ( string name )

Specifies the name of the heat pipe. Each heat pipe must have a unique name.

 

Folder ( name foldername )

Sets the name of the folder for the new heat pipe. If the name is empty, then the heat pipe does not belong to a folder.

 

Enable ( bool flag )

Enables or disables the heat pipe being created. If flag is false, the heat pipe will not be considered for simulation and the underlying geometry along with the material will be considered normally instead.

 

EnableItem( name  name,  bool flag )

Enables or disables the (existing) heat pipe with the given name depending on flag. If flag is false, the heat pipe will not be considered for simulation.

 

MaxAllowedHeatInWatt ( double expression Qmax  )

Defines the maximum heat (in W) allowed through the heat pipe. If this value is exceeded in the final solution, a warning message will be printed out.

 

ThermalResistance ( double expression resistance )

Sets the thermal resistance of the contacting solids.

 

ThermalResistanceUnit ( enum unit { "Absolute (K/W)", "Per unit area (K/W/m^2)" } )

Sets the thermal resistance to be defined either absolute or per unit area.

 

OverwriteDefaultThermalConductivity ( bool flag  )

When flag is True, the default thermal conductivity (50000 W/K/m) for the heat pipe is replaced by the user-defined value.

 

ThermalConductivity ( double expression conductivity )

Specifies the thermal conductivity value for the heat pipe. It is applied if the OverwriteDefaultThermalConductivity flag is set to True.

 

UseSpatiallyVaryingThermalConductivity( bool flag  )

When flag is True, the default thermal conductivity (50000 W/K/m) for the heat pipe is replaced by the user-defined space map based thermal conductivity.

 

AnisotropicSpatiallyVaryingThermalConductivity ( bool flag  )

When flag is True, the three components of the space map based thermal conductivity are considered.

 

ThermalConductivityMap ( string filename )

Sets the map .m3d format for the space map based thermal conductivity, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to False.

 

ThermalConductivityMapX ( string filename.m3d )

Sets the map in .m3d format for the space map based thermal conductivity in the "x" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True.

ThermalConductivityMap is an alias for ThermalConductivityMapX. It can be used instead.

 

ThermalConductivityMapY ( string filename.m3d )

Sets the map .m3d format for the space map based thermal conductivity in the "y" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True.

 

ThermalConductivityMapZ ( string filename.m3d )

Sets the map .m3d format for the space map based thermal conductivity in the "z" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True.

 

SpatiallyVaryingConductivityType (  enum type { "Analytical function", "File import" } )

Specifies the type of definition for the space map based thermal conductivity.

 

ThermalConductivityLaw ( double expression law )

Sets the law for the space map based thermal conductivity based on the SpatiallyVaryingConductivityType "Analytical function", when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to False. The law is evaluated at a discrete set of "x", "y", "z" coordinates. In this law, "x", "y" and "z" are used as variables. Parameters with those names are ignored when evaluating the law.

 

ThermalConductivityLawX ( double expression law )

Sets the law for the space map based thermal conductivity based on the SpatiallyVaryingConductivityType "Analytical function" in the "x" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The law is evaluated at a discrete set of "x", "y", "z" coordinates. In this law, "x", "y" and "z" are used as variables. Parameters with those names are ignored when evaluating the law.

ThermalConductivityLaw is an alias for ThermalConductivityLawX. It can be used instead.

 

ThermalConductivityLawY ( double expression law )

Sets the law for the space map based thermal conductivity based on the SpatiallyVaryingConductivityType "Analytical function" in the "y" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The law is evaluated at a discrete set of "x", "y", "z" coordinates. In this law, "x", "y" and "z" are used as variables. Parameters with those names are ignored when evaluating the law.

 

ThermalConductivityLawZ ( double expression law )

Sets the law for the space map based thermal conductivity based on the SpatiallyVaryingConductivityType "Analytical function" in the "z" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The law is evaluated at a discrete set of "x", "y", "z" coordinates. In this law, "x", "y" and "z" are used as variables. Parameters with those names are ignored when evaluating the law.

 

ThermalConductivityRangeXMin ( double expression value )

Sets the minimum "x" value for the evaluation of the law.

 

ThermalConductivityRangeXMax ( double expression value )

Sets the maximum "x" value for the evaluation of the law.

 

ThermalConductivityRangeYMin ( double expression value )

Sets the minimum "y" value for the evaluation of the law.

 

ThermalConductivityRangeYMax ( double expression value )

Sets the minimum "y" value for the evaluation of the law.

 

ThermalConductivityRangeZMin ( double expression value )

Sets the minimum "z" value for the evaluation of the law.

 

ThermalConductivityRangeZMax ( double expression value )

Sets the minimum "z" value for the evaluation of the law.

 

ThermalConductivitySamplesX ( int  value )

Sets the number of samples for the "x" coordinate for the evaluation of the law.

 

ThermalConductivitySamplesY ( int  value )

Sets the number of samples for the "y" coordinate for the evaluation of the law.

 

ThermalConductivitySamplesZ ( int  value )

Sets the number of samples for the "z" coordinate for the evaluation of the law.

 

ThermalConductivityImport ( string filename )

Sets the file to import for the space map based thermal conductivity for the SpatiallyVaryingConductivityType "File import", when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to False. The imported file must be specified in one of the formats supported by ConvertMaterialField, see Material Object.

 

ThermalConductivityImportX ( string filename )

Sets the file to import for the space map based thermal conductivity for the SpatiallyVaryingConductivityType "File import" in the "x" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The imported file must be specified in one of the formats supported by ConvertMaterialField, see Material Object.

ThermalConductivityImport is an alias for ThermalConductivityImportX. It can be used instead.

 

ThermalConductivityImportY ( string filename )

Sets the file to import for the space map based thermal conductivity for the SpatiallyVaryingConductivityType "File import" in the "y" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The imported file must be specified in one of the formats supported by ConvertMaterialField, see Material Object.

 

ThermalConductivityImportZ ( string filename )

Sets the file to import for the space map based thermal conductivity for the SpatiallyVaryingConductivityType "File import" in the "z" direction, when the AnisotropicSpatiallyVaryingThermalConductivity  flag is set to True. The imported file must be specified in one of the formats supported by ConvertMaterialField, see Material Object.

 

AddHeatPipePart ( solidname solidname, int faceid )

Adds a solid to the solid list for the heat pipe. The solid is picked by a face (indicated by its faceid) of a certain solid (indicated by its solidname).

 

AddContactingSolid ( solidname solidname, int faceid )

Adds a solid to the list of contacting solids for the heat pipe. The solid is picked by a face (indicated by its faceid) of a certain solid (indicated by its solidname).

 

Create

Creates the heat pipe with the previously set parameters.

 

Delete ( string name )

Deletes the heat pipe with the given name.

 

Rename  ( string oldname,  string newname  )

Renames the heat pipe with the given oldname to the defined newname.

 

NewFolder ( string foldername )

Creates a new folder with the given name.

 

RenameFolder ( string oldFoldername, string newFoldername)

Changes the name of an existing folder.

 

DeleteFolder ( string foldername )

Deletes an existing folder and all the contained elements.

Example

With HeatPipe

     .Reset

     .Name "heatpipe1"

     .Folder ""

     .Enable "True"

     .MaxAllowedHeatInWatt "4"

     .ThermalResistance "2"

     .ThermalResistanceUnit "Absolute (K/W)"

     .OverwriteDefaultThermalConductivity "False"

     .ThermalConductivity "55000"

     .AddHeatPipePart "component1:solid2", "2"

     .AddContactingSolid "component1:solid1", "12"

     .AddContactingSolid "component1:solid3", "3"

     .Create

End With
