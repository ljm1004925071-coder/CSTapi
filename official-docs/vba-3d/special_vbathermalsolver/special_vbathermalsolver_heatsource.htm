HeatSource Object

This object is used to create new heat sources for the Thermal Solver.

  • Methods

  • Default Settings

  • Example

Methods

Reset

Resets all internal settings of the heat source to their default values.

 

Name ( string name )

Specifies the name of the heat source.

 

Folder ( name foldername )

Sets the name of the folder for the new heat source. If the name is empty, then the heat source does not belong to a folder.

 

Enable( bool flag )

Sets the heat source to flag. If flag is false, the heat source won't be considered for simulation and the underlying geometry along with the material will be considered instead.

 

EnableItem( name  name,  bool flag )

Enables or disables the (existing) heat source with the given name depending on flag. If flag is false, the heat source will not be considered for simulation.

 

Value ( double expression value )

Specifies the heat flow value for the heat source.

 

ValueType ( enum {"Integral", "Density"} type )

Specify if the volume heat source value is defined as an integral type or a density type.

 

Face ( solidname solidname, int faceid )

A heat source can be defined on several faces. This method adds a face (indicated by its faceid) of a certain solid (indicated by its solidname) to the face list for the heat source.

 

Create

Creates the heat source with its previously made settings.

 

Delete ( string name )

Deletes the heat source with the given name.

 

Rename  ( string oldname,  string newname  )

Renames the heat source with the given oldname to the defined newname.

 

GetValue (name name )

Returns the value of a heat source with a given name.

 

GetType (name name )

Returns the type ("PTC" or "Volume") of a heat source with a given name.

 

GetValueType (name name )

Returns the value type ("Integral" or "Density") of a volume heat source source with a given name.

 

NewFolder ( string foldername )

Creates a new folder with the given name.

 

RenameFolder ( string oldFoldername, string newFoldername)

Changes the name of an existing folder.

 

DeleteFolder ( string foldername )

Deletes an existing folder and all the contained elements.

 

TemperaturePowerCurveActive ( bool  flag )

Activates or deactivates the temperature dependent heat source.

 

ResetNLPower

Resets stored parametrization data.

 

AddNLPowerValue ( string temperature, string power)

Adds a temperature-power pair value to the parametrization.

 

TemperatureUnitCurve ( string TemperatureUnit)

Sets the desired temperature unit.

 

PowerUnitCurve ( string PowerUnit)

Sets the desired power unit.

 

TemperatureDependencyType ( string TempDepType)

Selects how the power will be calculated for the particular object. Options are "Average" and "Maximum".

Default Settings

Name ("")

Value ("")

Face ("", 0)

Type ("PTC")

Enable (True)

Folder("")

TemperaturePowerCurveActive(False)

TemperatureUnitCurve("K")

PowerUnitCurve("W")

TemperatureDependencyType("Average")

Example

With HeatSource

     .Reset

     .Name "heatcurrent1"

     .Type "PTC"

     .Value "100"

     .Face "component1:solid1", "1"

     .TemperaturePowerCurveActive "1"

     .ResetNLPower

     .AddNLPowerValue "280", "2.3"

     .AddNLPowerValue "300", "3.6"

     .AddNLPowerValue "330", "5.3"

     .TemperatureUnitCurve "K"

     .PowerUnitCurve "W"

     .TemperatureDependencyType "Average"

     .Create

End With
