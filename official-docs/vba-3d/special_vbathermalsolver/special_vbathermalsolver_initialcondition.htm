InitialCondition Object

This object is used to specify initial conditions for solids and fluid domains for the Conjugate Heat Transfer Solver.

For solids, the initial temperature can be specified. For fluid domains the initial fluid velocity and/or temperature can be specified.

Methods

Reset

Resets all internal settings of the initial condition object to their default values.

 

Name ( string name )

Specifies the name of the initial condition. Each initial condition must have a unique name.

 

Set ( string parameterName, variant parameterValues, ... )

To define special parameters for a thermoelectric cooler, the command Set can be used. Available parameterNames are listed below. All values must be set before the Create command is called.

parameterName

parameterValues

Meaning

"Temperature"

 enum type { "Ambient", "Fixed" }

Initial temperature for the fluid domain or solid. The second parameter specifies the way in which the temperature is prescribed. Available types are:

"Ambient" The ambient temperature, which is set in the solver parameter settings, will be used for the initial condition.
"Fixed" A user-defined fixed value can be assigned to the initial condition. In this case two further parameters are required: string value, enum unit, where unit is either "Kelvin" or "Celsius" or "Fahrenheit".

"Velocity"

double expression vx, double expression vy, double expression vz, enum unit

Initial velocity prescribed in the fluid domain. The unit can be either "m/s" or "km/h".

 

 

AddItem ( string objectname, string initialconditionname )

Adds the initial condition named initialconditionname to the object (fluid domain or solid) with the given objectname. The objectname must be prefixed by either "fluiddomain$" or "solid$", depending on the type of the object.

 

RemoveItem ( string objectname, string initialconditionname )

Removes the initial condition named initialconditionname from  the object (fluid domain or solid) with the given objectname. The objectname must be prefixed by either "fluiddomain$" or "solid$", depending on the type of the object.

 

Create

Creates the initial condition with the previously set parameters.

 

Delete ( string name )

Deletes the initial condition with the given name.

 

Rename  ( string oldname,  string newname  )

Renames the initial condition with the given oldname to the defined newname.

 

Example

With InitialCondition

     .Reset

     .Name "initialcondition1"

     .Set "Temperature", "Ambient"

     .Set "Velocity",0.0,0.0,0.0, "m/s"

     .Create

     .AddItem "solid$component1:solid1", "initialcondition1"

End With
