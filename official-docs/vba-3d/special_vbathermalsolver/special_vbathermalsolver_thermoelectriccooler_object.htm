ThermoelectricCooler Object

This object is used to specify a thermoelectric cooler for the Conjugate Heat Transfer Solver.

  • Methods

  • Example

Methods

Reset

Resets all internal settings of the thermoelectric cooler to their default values.

 

Name ( string name )

Specifies the name of the thermoelectric cooler. Each thermoelectric cooler must have a unique name.

 

Folder ( name foldername )

Sets the name of the folder for the new thermoelectric cooler. If the name is empty, then the thermoelectric cooler does not belong to a folder.

 

Enable ( bool flag )

Enables or disables the thermoelectric cooler being created. If flag is false, the thermoelectric cooler will not be considered for simulation and the underlying geometry along with the material will be considered normally instead.

 

EnableItem( name  name,  bool flag )

Enables or disables the (existing) thermoelectric cooler with the given name depending on flag. If flag is false, the thermoelectric cooler will not be considered for simulation.

 

AddColdSurface ( solidname solidname, int faceid  )

Defines the cold surface of the thermoelectric cooler. This will be the face belonging to the specified faceid of the solid with name solidname.

 

AddHotSurface ( solidname solidname, int faceid  )

Defines the hot surface of the thermoelectric cooler. This will be the face belonging to the specified faceid of the solid with name solidname.

 

Set ( string parameterName, variant parameterValues, ... )

To define special parameters for a thermoelectric cooler, the command Set can be used. Available parameterNames are listed below. All values must be set before the Create command is called.

parameterName

parameterValues

Default

Meaning

"HotSurfaceTemperature"

double expression temperature1, double expression temperature2

<empty / invalid>

Reference temperatures at hot side for the thermoelectric cooler performance characteristics.

"QmaxAtColdSurface"

double expression Qmax1, double expression Qmax2

<empty / invalid>

Maximum heat that can be absorbed at the cold side at the two reference temperatures.

"dTmaxFromHotToCold"

double expression dTmax1, double expression dTmax2

<empty / invalid>

Maximum temperature difference obtained between the hot and cold side under no heat load at the two reference temperatures.

"InputCurrentAtdTmax"

double expression Imax1, double expression Imax2

<empty / invalid>

Input current resulting in the maximum temperature difference at the two reference temperatures.

"VoltageAtdTmax"

double expression Vmax1, double expression Vmax2

<empty / invalid>

Voltage at the maximum temperature difference at the two reference temperatures.

"MaxAllowedTemperatureAtHotSurface"

double expression Tmax

<empty / invalid>

Maximum hot side temperature allowed at the hot side.

"OperationalCurrent"

double expression I

<empty / invalid>

The electrical current used to power the thermoelectric cooler for the current simulation.

"ThermalResistanceAtColdSurface"

bool flag

double resistance

 enum type { "Absolute", "Per unit area" }

False

0.0

"Absolute"

 When enabled (flag = True), the user-defined resistance is used to specify the thermal resistance at the cold surface. The thermal resistance can be entered as absolute (K/W) (type = "Absolute") or per unit area (K/W/m^2) (type = "Per unit area").

"ThermalResistanceAtHotSurface"

bool flag

double resistance

enum type { "Absolute", "Per unit area" }

False

0.0

"Absolute"

 When enabled (flag = True), the user-defined resistance is used to specify the thermal resistance at the hot surface. The thermal resistance can be entered as absolute (K/W) (type = "Absolute") or per unit area (K/W/m^2) (type = "Per unit area").

 

 

Create

Creates the thermoelectric cooler with the previously set parameters.

 

Delete ( string name )

Deletes the thermoelectric cooler with the given name.

 

Rename  ( string oldname,  string newname  )

Renames the thermoelectric cooler with the given oldname to the defined newname.

 

NewFolder ( string foldername )

Creates a new folder with the given name.

 

RenameFolder ( string oldFoldername, string newFoldername)

Changes the name of an existing folder.

 

DeleteFolder ( string foldername )

Deletes an existing folder and all the contained elements.

Example

With ThermoelectricCooler

     .Reset

     .Name "thermoelectriccooler1"

     .Set "HotSurfaceTemperature", "22", "45"

     .Set "QmaxAtColdSurface", "30", "34"

     .Set "dTmaxFromHotToCold", "74", "85"

     .Set "InputCurrentAtdTmax", "3.5", "3.5"

     .Set "VoltageAtdTmax", "16.0", "18.0"

     .Set "MaxAllowedTemperatureAtHotSurface", "80"

     .Set "OperationalCurrent", "2.25"

     .Set "ThermalResistanceAtColdSurface", "False", "0.4", "Absolute"

     .Set "ThermalResistanceAtHotSurface", "False", "0.15", "Absolute"

     .AddColdSurface "component1:solid1", "2"

     .AddHotSurface "component1:solid1", "1"

     .Create

End With
