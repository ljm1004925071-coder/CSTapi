General Methods

Create

Creates a new probe. All necessary settings (a connected block pin) have to be made previously.

 

Delete

Deletes a probe. All necessary settings (probe name) have to be made previously.

 

DoesExist bool

Checks if a probe with the currently defined properties (probe name) already exists.

 

Reset

Resets all internal settings to their default values.

Identification

Name ( name probename )

Sets the name of a probe before calling Create. Furthermore, this method can be used to select an existing probe of your model prior of calling queries.

Getter

GetName name

Returns the name of the probe with the currently defined properties.

 

GetType int

Returns the type of the probe with the currently defined properties. The following int values are returned:

0: No probe

1: normal probe

2: differential probe

Setter

SetName ( name probename )

Modifies the name of an existing probe.

 

SetBlockPin ( string type, name componentname, name pinname, bool directioninwards )

Sets the probe's location by the specification of the component pin to be connected to. Furthermore, the probe's direction must be given (probe points to the component's pin if true).

See table below for description of arguments.

 

SetRefBlockPin ( string type, name componentname, name pinname )

Sets the differential probe's reference location by the specification of type, componentname and pinname.

 

Component

type

componentname

pinname

Block

"B"

Name of the block as displayed in the tree.

Name of the pin as displayed in the schematic. For blocks without visible pin names (e.g. resistors or transistors) use the names that are displayed near the block icon in the online help of the corresponding block, or numbers starting with 1 if the icon does not contain pin names.

External Port

"P"

Name of the external port as displayed in the tree.

Empty string to choose the main pin or "ref" (case insensitive) to choose the reference pin (only valid for differential external ports).

Iteration

StartProbeNameIteration int

Resets the iterator for the probes and returns the number of probes.

 

GetNextProbeName name

Returns the next probe's name. Call StartProbeNameIteration before the first call of this method.

Direction

NormalDirection bool

Returns True if the direction of the probe with the currently defined properties is normal.

 

ChangeDirection

Changes a probe's direction, i.e. the current is considered with opposite sign.

Examples

'Create a new normal probe

With CircuitProbe

.Reset

.Name ("Probe1")

.SetBlockPin("P", "1", "", TRUE) ' define probe location: external port 1, pointing towards the port

.Create ' Create normal probe 'Probe1'

End With

 

'Create a new differential probe

With CircuitProbe

.Reset

.Name ("DiffProbe")

.SetBlockPin("P", "2", "REF", TRUE) ' define probe location: external port 2, differential pin, pointing towards the port

.SetRefBlockPin("B", "RES1", "2") ' define reference probe location: block RES1-Pin2

.Create ' Create differential probe 'DiffProbe' between port 2 differential pin and block RES1-Pin2

End With

 

'Modify an existing probe

With CircuitProbe

.Reset

.Name ("Probe1")

.SetName ("NormalProbe") ' Rename

.ChangeDirection ' Change probe direction

End With
