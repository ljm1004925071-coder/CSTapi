General Methods

Create

Creates a new external port.

 

Delete

Deletes the currently selected external port.

 

DoesExist bool

Checks if an external port with the currently selected number already exists.

 

Reset

Resets all internal settings to their default values.

Identification

Name ( name portname )

This method is used to select an existing external port of your model using its name. Furthermore, it sets the port name to portname if not created yet.

Getter

GetLabel string

Returns the external port's label. If no label is assigned to the considered external port, an error message with the port number is thrown.

 

GetPortName( int portindex ) string

Returns the external port's inne port name referenced by the given portindex.

 

GetNumberOfPorts int

Returns the external port's bus size, which corresponds to the number of inner ports.

 

IsDifferential bool

Returns true if the external port has a differenital pin.

 

HasCommonReference bool

Returns true if the external port has common reference pin.

Setter

SetName ( name portname )

Modifies the name of an existing external port.

 

SetLabel ( string portlabel )

Modifies or sets a label to an existing external port.

 

SetPortName ( int portindex, name portname )

Modifies the name of an inner port at the given portindex.

 

SetNumberOfPorts ( int numberOfPorts )

Sets the external port's bus size, which corresponds to the number of inner ports.

 

SetDifferential ( bool differential )

Makes the external port a differential port, i.e. its reference pin can be accessed.

 

SetCommonReference ( bool commonReference )

Unifies all reference ports of the differential bus pin into a signle port.

Iteration

StartPortNameIteration int

Resets the iterator for external ports and returns the number of external ports.

 

GetNextPortName name

Returns the next external port's name. Call StartPortNameIteration before the first call of this method.

Positioning

GetPositionX int

Returns the horizontal position of the center point of an external port.

 

GetPositionY int

Returns the vertical position of the center point of an external port.

 

Position ( string x, string y )

Specifies the position of the external port. This setting must be made before calling Create. Furthermore, it can be used to modify an external port's position. Possible values are 0 < x, y <100000, i.e. (50000, 50000) is the center of the schematic. It is always ensured that the external port is aligned with the grid, therefore the specified position might get adjusted slightly.

 

Rotate ( string angle )

Rotates the external port by the given angle in degrees around its center point. If this setting is made before calling Create, the angle is only stored. The rotation will then be done in Create.

 

Move ( string x, string y )

Moves an external port by the given offset. Note that the schematic size is given by 0 < x, y <100000. Use Position to specify a certain location. It is always ensured that the external port is aligned with the grid, therefore the specified offset might get adjusted slightly.

Default Settings

Position (50000, 50000)

Examples

'Create an external port

 

With ExternalPort

.Reset

.Name ("1")

.Position(50000, 50000)

.Create

.SetLabel("First port")

End With

 

'Modify an existing external port's position

 

With ExternalPort

.Reset

.Name ("1")

.Position(57000, 50500) ' Move to absolute position (57000, 50500)

.Move(7000, 500) ' Move by offset (7000, 500) relative to old position

End With
