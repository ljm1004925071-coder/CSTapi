EyeAnalysisTask Object

Object referring to an eye analysis task. Use this object to manipulate an eye analysis task .

  • Methods

  • Example

Methods

Name (name taskname)

Sets the name of the eye analysis task to manipulate.

 

SetSourceTask(name sourcename)

Sets the name of the source task in the eye analysis task.

 

SetMode(string  mode)

Sets the eye generation mode of the eye analysis task. Available options are Transient and Statistical.

 

SetResponseType(string  type)

Selects  the type of the channel response  to be used in  the statistical eye generation of the eye analysis task. Available options are Step, Pulse, and Impulse.

 

SetParametric( string groupname, bool bactivate)

Activates or deactivates the parametric storage of the results for the given group. Supported values for the latter are "Eye diagram", "Mask analysis", and " Timing analysis" ; namely, the three property groups that appear in the task parameter list of the eye analysis task.   

 

SetUnitInterval (string UI_exp)

Sets the expression of the unit interval in the eye analysis task. If the eye analysis task's source task is an IbisAmi task, then the unit interval is not settable and its expression is readily obtained from that of the source  IbisAmi task.

 

SetDiagramCenter (string EDC_exp)

Sets the expression of the eye diagram center. This setting is only meaningful if the generation mode of the eye analysis task is Transient.  

 

SetStartTime (string expression)

Sets the expression of the start time, i.e. the time before which the source signal is excluded from the eye analysis. This setting is only meaningful if the generation mode of the eye analysis task is Transient.  

 

SelectProbe (bool bSelect, int index)

Selects or deselects the specified probe signal for wrapping into an eye diagram. Probe signal are only available if the eye analysis task's source task is a transient task.

 

SelectChannel (bool bSelect, int index)

Enable or disable the specified channel for eye diagram generation. Channels are only available if the eye analysis task's source task is an IbisAmi or a S-parameter task.

 

SelectChannelSection (bool bSelect, int index)

Selects or deselects the specified channel section for eye diagram generation. Channel sections are only available if the eye analysis task's source task is an IbisAmi task.

 

CountProbes int

Retrieves the number of probe signals available in the eye analysis task. If the eye analysis task's source task is an IbisAmi or a S-parameter task, the returned number is by definition zero.

 

CountChannels int

Retrieves the number of channels available in the eye analysis task. If the eye analysis task's source task is a transient task, the returned number is by definition zero.

 

CountChannelSections int

Retrieves the number of channel sections available in the eye analysis task. If the eye analysis task's source task is a transient or S-parameter task, the returned number is by definition zero.

 

GetSourceTask string

Gets the name of the source task in the eye analysis task.

 

GetMode string

Gets the eye generation mode in the eye analysis task.

 

GetResponseType string

Gets the channel response type set for statistical eye generation. An error is displayed if the eye analysis task's source task is a S-parameter task or the generation mode is Transient.

 

GetParametric(string groupname) bool

Checks if the parametric storage of the results for the given group is activated or deactivated. Supported values for the latter are "Eye diagram", "Mask analysis", and " Timing analysis" ; namely, the three property groups that appear in the

task parameter list of the eye analysis task.

 

GetUnitInterval string

Gets the expression of the unit interval in the eye analysis task.

 

GetDiagramCenter string

Gets the expression of the diagram center in the eye analysis task. If the generation mode of the eye analysis task is Statistical, the returned value is "0." by definition.   

 

GetStartTime string

Returns the expression specifying the start time of the eye analysis. If the generation mode of the eye analysis task is Statistical, the returned value is "0." by definition.   

 

GetProbe (int index) string

Gets the name of the specified probe in the eye analysis task. If the eye analysis task's source task is an IbisAmi or a S-parameter task, the returned string is by definition empty.

 

GetProbeIndex (string name) int

Gets the probe index associated with the specified probe signal in the eye analysis task. If the eye analysis task's source task is an IbisAmi or a S-parameter task, an error is displayed.

 

DoesProbeExist (string name) bool

Checks if the specified probe signal exists in the eye analysis task. If the eye analysis task's source task is an IbisAmi or a S-parameter task, the returned value is by definition false.

 

GetChannel (int index) string

Gets the name of the specified channel in the eye analysis task. If the eye analysis task's source task is a transient task, the returned string is by definition empty.

 

GetChannelSection (int index) string

Gets the name of the specified channel section in the eye analysis task. If the eye analysis task's source task is a transient or a S-parameter  task, the returned string is by definition empty.

 

IsProbeSelected (int index) bool

Checks if the specified probe signal has been selected for wrapping into an eye diagram. If the eye analysis task's source task is an IbisAmi or a S-parameter task, the returned value is by definition false.

 

IsChannelSelected (int index) bool

Checks if the specified channel has been included for eye diagram generation. If the eye analysis task's source task is a transient task, the returned value is by definition false.

 

IsChannelSectionSelected (int index) bool

Checks if the specified channel section has been selected for eye diagram generation. If the eye analysis task's source task is a transient or S-parameter task, the returned value is by definition false.

 

The following functions are used to set up and query the attributes of the eye mask in the eye analysis task.

 

SetMask (string shape, variant Atts)

Sets the eye mask in the eye analysis task. Possible values for shape are "None", "Rectangle", "Trapezoid", "Cross", "Symmetric hexagon", and "Asymmetric hexagon" while Atts is an array containing the expressions defining the mask parameters as defined in the eye analysis task.

 

SetMaskAttribute (string Att, string Att_exp)

Sets the expression of the specified mask attribute. Possible values for Att can be "Shape" to set the mask shape, and any of the mask parameters as defined in the eye analysis task. When setting for the mask shape, valid strings for Att_exp are the ones mentioned in the description of the SetMask command. If the shape of the mask is set to "None", then setting any valid mask parameter has no effect.

 

GetMaskShape string

Gets the shape of the eye mask in the eye analysis task. Possible returned strings are the mask shapes mentioned in the description of the SetMask command.

 

GetMaskAttribute (string Att) string

Gets the expression of the specified mask attribute in the eye analysis task. Possible values for Att can be "Shape" to return the mask shape or any of the mask parameters as defined in the eye analysis task. If the shape of the mask is set to "None", then querying for any valid mask parameter returns always an empty string by definition.

 

EnableEyeCalculation (bool b1DEye, bool b2DEye)

Enables or disables the type of eye diagram to generate by the eye analysis task. If b1DEye is enabled then the diagrams are generated by wrapping probe signals into a standard eye pattern. On the other hand, if b2DEye is enabled then the diagrams are generated by wrapping probe signals into a 2D histogram displayed as a colormap representing the frequency with which the signals cross the histogram bins.

 

SetAutomaticMaskCentering(string option)

Selects an option for the mask placement. Options are 'Enabled', 'Disabled', 't_ref only', and 'V_ref only'. If 'Enabled' is chosen, an algorithms places the mask into the eye so that mask violations are avoided - if possible. If 'Disabled' is chosen, the mask is placed according to the t_ref and V_ref settings. If 't_ref only' or 'V_ref only' is chosen, an algorithm optimizes the mask position only in the time or voltage dimension, respectively.

 

GetAutomaticMaskCentering string

Tells the chosen mask placement option. Possible return values are the valid options of the SetAutomaticMaskCentering command.

 

ShowMask (bool bshow)

Enables or disables the display of the defined eye mask in the eye diagram plot.

 

IsMaskShown bool

Check if the display of the defined eye mask in the eye diagram plot is enabled.

 

The following functions are used to set up and query the eye diagram attributes.

 

Is1DEnabled bool

Checks if the standard eye generator is enabled in the eye analysis task .

 

Is2DEnabled bool

Checks if the histogram-based eye generator is enabled in the eye analysis task .

 

SetTimeResolution (string resolution)

Sets the resolution of 2D eye diagrams in the time dimension. Valid arguments are expressions evaluating to positive numbers.

 

GetTimeResolution string

Gets the expression of the time resolution in the eye analysis task.

 

SetVoltageResolution (string resolution)

Sets the vertical height of the bins in the histogram-based eye generator of the eye analysis task. Permissible values for resolution are positive-valued expressions.

 

GetVoltageResolution string

Gets the expression of the voltage resolution in the eye analysis task.

 

The following functions are used to set up and query the timing analysis configuration in the eye analysis task.

 

 SetGroupSpreadAnalysis (bool enable)

 Enables or disables the group spread calculation in the eye analysis task.

 

GetGroupSpreadAnalysis bool

Checks if the group spread calculation is enabled in the eye analysis task .

 

 SetRefDistanceAnalysis (bool enable)

 Enables or disables the reference distance calculation in the eye analysis task. This setting is meaningful if the generation mode of the eye analysis task is Transient; otherwise, an error is displayed.

 

GetRefDistanceAnalysis bool

Checks if the reference distance calculation is enabled in the eye analysis task .

 

SetRefSignal (string name)

Sets the name of the signal that will serve as the timing reference in the reference distance calculation, e.g. "P1 Diff " for a differential probe "P1". Only differential signals can be selected as reference. This setting is meaningful if the generation mode of the eye analysis task is Transient; otherwise, an error is displayed.

 

GetRefSignal string

Gets the path of the reference signal in the reference distance calculation.

 

SetFrequencyRatio (string ratio)

Sets the ratio of the reference signal's frequency to the data signal's frequency for timing analyses . Possible values for ratio are "1", "2", and "4". For information on the ratios see the description of the eye analysis task. This setting is meaningful if the generation mode of the eye analysis task is Transient; otherwise, an error is displayed.

 

GetFrequencyRatio string

Returns the specified frequency ratio. Possible return values are "1", "2", and "4".

 

SetNominalOffset (string offset)

Sets the nominal offset to be taken into account in the reference distance calculation. The reference signal is virtually shifted by that offset for the calculation of the distance. Options for offset are "None" and "Half unit interval". This setting is meaningful if the generation mode of the eye analysis task is Transient; otherwise, an error is displayed.

 

GetNominalOffset string

Tells the selected nominal offset.

 

 SetGroupSpreadAnalysis (bool enable)

Enables or disables the extent analysis.

 

GetExtentAnalysis bool

Tells whether the extent calculation is enabled.

Example

'Create an eye analysis task nested in an IbisAmi task

 

With SimulationTask

.Reset

.Type ("IbisAmi")

.Name ("IbisAmi1")

.Create

.Reset

.Type ("Eye Analysis")

.Name ("IbisAmi1\EyeAnalysis1")

.Create

End With

 

'Setting Eye diagram center

 

With EyeAnalysisTask

.Name ("IbisAmi1\EyeAnalysis1")

.SetDiagramCenter ("0.01")

.SetMaskAttribute ("Shape", "Rectangle")

End With
