IbisAmiTask Object

Object referring to an IbisAmi task. Use this object to manipulate an IbisAmi task .

  • Methods

  • Example

Methods

Name (name taskname )

Sets the name of the IbisAmi task to manipulate.

 

SetUnitInterval (string UI_exp)

Sets the expression of the unit interval in the IbisAmi task.

 

SetMode (string mode )

Sets the simulation mode of the IbisAmi task. Valid choices are: "Statistical" or "Transient".

 

SetNumberOfBits (string Nbits)

Sets the number of bits to simulate in the IbisAmi task. This parameter is relevant if mode is set to "Transient".

 

AddLink (string Tx, string Rx, string Channel)

Creates a new AMI link in the IbisAmi task.

Tx: Specifies the name of a transmitter IBIS buffer.

Rx: Specifies the name of a receiver IBIS buffer.

Channel: Specifies the name of the transient or AC task that simulates the analog channel linking the IBIS transmitter and receiver buffers.

 

DeleteLink (int index)

Deletes the specified AMI link from the IbisAmi task.

 

SelectCrossTalk(int iaggressor, int ivictim)

Select a new cross-talk link in the IbisAmi task when mode is set to "Transient". If mode is set to "Statistical" the function has no effect.

iaggressor: Index of the AMI link that becomes the aggressor.

ivictim: Index of the AMI link that becomes the victim.       

 

SelectAllCrossTalks(bool bSelect)

Select or deselect all cross-talk links in the IbisAmi task when mode is set to "Transient". If mode is set to "Statistical" the function has no effect.

 

CountLinks int

Retrieves the number of existing AMI links in the IbisAmi task.

 

GetUnitInterval string

Gets the expression of the unit interval in the IbisAmi task.

 

GetMode string

Gets the simulation mode of the IbisAmi task. Possible returned strings are "Statistical" or "Transient".

 

GetNumberOfBits string

Gets the number of bits in the IbisAmi task. This parameter is relevant if mode was set to "Transient".

 

GetTransmitter (int index) string

Gets the name of the transmitter IBIS buffer from the specified AMI link in the IbisAmi task.

 

GetReceiver (int index) string

Gets the name of the receiver IBIS buffer from the specified AMI link in the IbisAmi task.

 

GetChannel (int index) string

Gets the name of the transient or AC task that simulates the analog channel in the specified AMI link of the IbisAmi task.

 

HasAggressor(int ivictim) bool

Tells if the specified AMI link has aggressors.

 

HasVictim(int iaggressor) bool

Tells if the specified AMI link has victims.

 

GetAggressors(int ivictim) variant

Gets the aggressors of the specified victim AMI  link.

 

GetVictims(int iaggressor) variant

Gets the victims of the specified aggressor AMI  link.

Example

'Creation of an IbisAmi task

 

With SimulationTask

.Reset

.Type("IbisAmi")

.Name ("IbisAmi1")

.Create

End With

 

'Setting some parameters of the IbisAmi task

 

With IBISAMITask

.Name ("IbisAmi1")

.SetUnitInterval ("2")

.SetMode ("Transient")

.SetNumberOfBits ("1000")

End With
