ArrayTask Object

Object referring to an array task. Use this object to create or to manipulate an array task.

  • Methods Concerning Existing Array Tasks

  • Queries

  • Example

Methods Concerning Creation of Array Task

Reset

Resets all internal settings to their default values.

 

Name ( name taskname )

Sets the name of a phased array antenna task before calling Create.

 

SetExcitationPattern (string pattern)

Sets the excitation pattern for the phased array antenna. Valid choices are: Uniform, Cosine, Cosine-Sq., Taylor, Binomial, Chebyshev and Multiple Excitations.

 

SetElementModel (name elementmodel)

Sets the element model for the phased array antenna.

 

SetTSVFilename(string filename)

Sets the TSV filename and uses the data in the file to define the phased array antenna task.

 

SetShapeCustom (string elements_in_x, string elements_in_y, string spacing_in_x, string spacing_in_y, string grid_angle)

Sets the settings for creating a custom shape phased array antenna

elements_in_x: Specify the number of elements in the X direction.

elements_in_y: Specify the number of elements in the Y direction.

spacing_in_x: Specify the spacing in the X direction. This is the distance from an element's center to the center of an adjacent element in the X direction.

spacing_in_y: Specify the spacing in the Y direction. This is the distance from an element's center to the center of an adjacent element in the Y direction.

grid_angle: Specify the grid angle of the array.

 

SetShapeDiamond (string elements_in_a_side, string spacing_in_x, string spacing_in_y)

Sets the settings for creating a diamond shape phased array antenna

elements_in_a_side: Specify the number of elements in one side of the diamond.

spacing_in_x: Specify the spacing in the X direction. This is the distance from an element's center to the center of an adjacent element in the X direction.

spacing_in_y: Specify the spacing in the Y direction. This is the distance from an element's center to the center of an adjacent element in the Y direction.

 

SetShapeHexagonal (string elements_in_a_side, string spacing_in_x, string spacing_in_y)

Sets the settings for creating a hexagonal shape phased array antenna

elements_in_a_side: Specify the number of elements in one side of the hexagon.

spacing_in_x: Specify the spacing in the X direction. This is the distance from an element's center to the center of an adjacent element in the X direction.

spacing_in_y: Specify the spacing in the Y direction. This is the distance from an element's center to the center of an adjacent element in the Y direction.

 

SetShapeOctagonal (string elements_in_a_side, string spacing_in_x, string spacing_in_y)

Sets the settings for creating a octagonal shape phased array antenna

elements_in_a_side: Specify the number of elements in one side of the octagon.

spacing_in_x: Specify the spacing in the X direction. This is the distance from an element's center to the center of an adjacent element in the X direction.

spacing_in_y: Specify the spacing in the Y direction. This is the distance from an element's center to the center of an adjacent element in the Y direction.

 

Create

Creates a new phased array antenna task.

Methods Concerning Existing Array Tasks

AddTSVFilenameToMultipleExcitationList(name taskname, string filename)

Adds a TSV file to the multiple excitation lists of an existing array task. The array excitation pattern will be set to Multiple Excitations.

 

ImportAmplitudeAndPhaseValuesFromTSVFilename(name taskname, string filename)

Imports the amplitude and phase value of a TSV file in an existing array task.

 

SetElementExcitationType(name taskname, string type)

Sets the element excitation type of an existing array task. Valid choices are: Simultaneous and Individual.

 

SetPort(name taskname, string portNumber, string modeNumber, bool activeflag, string amplitude, string phaseshift)

Sets the parameters of a port in the element excitations of an existing array task. The port is specified via the portnumber and modenumber. The following parameters are set in the port: activeflag, amplitude and phaseshift.

Queries

GetElementModel(name taskname) name

Returns the name of the element model of the array task.

 

GetTSVFilename(name taskname) name

Returns the TSV filename of the array task.

 

GetElementExcitationType(name taskname) name

Returns the element excitation type of the array task.

 

GetAmplitudeStringValues(name taskname) variant

Returns an array of the amplitude values as strings of the array task.

 

GetPhaseStringValues(name taskname) variant

Returns an array of the phase values as strings of the array task.

Example

' Example 1

'Delete the task if it exists

 

With SimulationTask

.Reset

.Name ("Task1")

If .doesexist Then .delete

End With

 

'Create a new phased array antenna task

 

With PhasedArrayTask

.Reset

.Name ("Task1")    

.SetShapeCustom ("5", "5", "10.0", "10.0", "63.43")

.SetExcitationPattern ("Cosine")

.Create

End With

 

'Import amplitude and phase values of a TSV-file to the created array task

 

PhasedArrayTask.ImportAmplitudeAndPhaseValuesFromTSVFilename("Task1", "import_ampl_phase\Model\3D\5by5.tsv")

 

'Get the amplitude values from the array task

 

Dim amplitudeValues As Variant

PhasedArrayTask.GetAmplitudeStringValues("Task1", amplitudeValues)

 

Dim amplVal0 As String

amplVal0 = amplitudeValues(0)

 

' Example 2

'Delete the task if it exists

 

Dim sArrayTaskName As String

sArrayTaskName = "Task2"

 

With SimulationTask

.Reset

.Name sArrayTaskName

If .doesexist Then .delete

End With

 

'Create a new phased array antenna task using a TSV-file which is located next to the CST-file

 

With PhasedArrayTask

.Reset

.Name sArrayTaskName     

.SetElementModel ("Block: MWSSCHEM1")

 .SetExcitationPattern ("Multiple Excitations")

 .SetTSVFilename ("MultipleLists\Array_Layout.tsv" )

.Create

End With

 

'Specify the element excitation type and the port parameters of the created array task

 

PhasedArrayTask.SetElementExcitationType(sArrayTaskName, "Individual")

PhasedArrayTask.SetPort(sArrayTaskName, "1", "", True, "1.0", "0.0")

PhasedArrayTask.SetPort(sArrayTaskName, "2", "", True, "1.0", "0.0")

 

'Add TSV-files to the multiple excitation lists of the array task

 

PhasedArrayTask.AddTSVFilenameToMultipleExcitationList(sArrayTaskName, "MultipleLists\lists\1_Uniform.tsv")

PhasedArrayTask.AddTSVFilenameToMultipleExcitationList(sArrayTaskName, "MultipleLists\lists\2_TaylorTaper.tsv")

PhasedArrayTask.AddTSVFilenameToMultipleExcitationList(sArrayTaskName, "MultipleLists\lists\3_BroadsideNull.tsv")

PhasedArrayTask.AddTSVFilenameToMultipleExcitationList(sArrayTaskName, "MultipleLists\lists\4_FanBeam.tsv")
