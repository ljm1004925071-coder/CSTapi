Methods for Units

SetUnit (enum dimension, enum unit)

Sets the entered unit for the selected dimension. Admissible values for dimension and unit are:

dimension

supported units

Length

nm, um, mm, cm, m, mil, in, ft

Time

fs, ps, ns, us, ms, s

Frequency

Hz, kHz, MHz, GHz, THz, PHz

Temperature

degC, K, degF

Voltage

nV, uV, mV, V ,kV, MV

Current

nA, uA, mA, A, kA, MA

Resistance

uOhm, mOhm, Ohm, kOhm, MOhm, GOhm

Conductance

nS, uS, mS, S

Capacitance

fF, pF, nF, uF, mF, F

Inductance

pH, nH, uH, mH, H

 

Unit strings are case-sensitive and follow the syntax described in the units documentation.

 

GetUnit (enum dimension) string

Returns the current unit setting for dimension. See SetUnit for more information on the returned unit strings.

Methods for Geometric Units

GetGeometryUnitToSI double

Returns the factor to convert a geometry value measured in units of the current project into the units.

-> a(SI Unit) = factor * b(project Unit)

 

GetGeometrySIToUnit double

Returns the factor to convert a geometry value in SI units into the units defined in the current project.

-> a(project Unit) = factor * b(SI Unit)

Methods for Result Units

SetResultUnit ( string Quantity, string displayAs, string Unit )

This method is available in 3D projects only. It sets a global default display unit for Quantity. The configured unit is used by 1D plots, 2D and 3D plots.

Supported quantities and display units are:

Quantity

display as

Dimension

"frequency"

"frequency"

frequency

 

"wavelength"

length

 

"wavenumber"

1/length

Any legal Unit with dimension Dimension is supported. Please refer to the documentation of the unit syntax for further information.

Example

With Units

     .SetUnit ("Length", "mm")

     .SetUnit ("Frequency", "GHz")

     .SetUnit ("Time", "s")

     .SetUnit ("Voltage", "mV")

     .SetUnit ("Current", "uA")

     .SetUnit ("Resistance", "kOhm")

     .SetUnit ("Conductance", "mS")

     .SetUnit ("Capacitance", "pF")

     .SetUnit ("Inductance", "nH")

     .SetUnit ("Temperature", "degC")

End With
