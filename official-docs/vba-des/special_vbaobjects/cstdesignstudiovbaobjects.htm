CST Design Studio VBA Objects

CST Studio Suite is implemented as an OLE automation server. The application is registered under "CSTStudio.Application". An instance can be created through the CreateObject VBA command:

Dim app As Object

Set app = CreateObject ("CSTStudio.Application")

This application object provides methods to open and create projects. Refer to the Application Object for details.

If you open the built-in VBA-Interpreter for a CST Design Studio project all of these objects are already available. To call the method AMethod of the object AnObject simply type AnObject.AMethod ( Parameters ). All functions and methods of the Project Object and the Units Object can be called directly, because you are in the scope of this object.
